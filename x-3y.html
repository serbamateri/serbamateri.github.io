13. x² (2x² + x + 3y) -4 (2x² + x + 3y) ​


Penjelasan dengan langkah-langkah:

Ekspresi yang diberikan adalah:

x²(2x² + x + 3y) - 4(2x² + x + 3y)

Untuk menyederhanakan ekspresi ini, kita bisa faktorkan keluar 2x² + x + 3y dari kedua suku:

(2x² + x + 3y)(x² - 4)

Sekarang, kita bisa faktorkan x² - 4 menggunakan perbedaan kuadrat:

(2x² + x + 3y)(x + 2)(x - 2)

Jadi, ekspresi yang telah disederhanakan adalah:

(2x² + x + 3y)(x + 2)(x - 2)
