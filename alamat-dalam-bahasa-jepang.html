27. Perkenalkan diri anda secara lengkap (nama, alamat,sekolah) dalam bahasa Jepang! ​


Perkenalkan diri anda secara lengkap (nama, alamat,sekolah) dalam bahasa Jepang!

JAWAB :自己紹介 (Jikoshoukai)  Perkenalan Diri dalam Bahasa Jepang

1. はじめまして (Hajimemashlte)    

= Perkenalkan / salam kenal

2. 私の名前は ... です (Watashi no namae wa ... desu)    

= Nama saya ...

3. ... と呼んでください ( ... to yonde kudasai)    

= Tolong panggil saja ...

4. 私は ... 歳です (Watashi wa ... sai desu)    

= Saya ... tahun

5. 私は ... ... 年生です (Watashi wa ... ... nensei desu)  

= Saya kelas ... ...

6. 私は ... に住んでいます (Watashi wa ... ni sunde imasu)    

= Saya tinggal di ...

7. 私の趣味は ... です (Watashi no shumi wa ... desu)    

= Hobi saya adalah ...

8. どうぞよろしくお願いします (Douzo yoroshiku onegaishimasu)    

= Senang berkenalan dengan anda

Contoh :  

はじめまして。私の 名前は アズカヒルミです。アズカと 呼んでください。私は1 5歳です。高校１年生です。私は 東京に 住んでいます。趣味は アニメを 見ることです。どうぞ よろしくお願いします。  

Hajimemashlte. Watashi no namae wa Azuka Hirumi desu. Azuka to yonde kudasai. Watashi wa 15-sai desu. Koukou 1-nensei desu. Watashi wa Toukyou ni sunde imasu. Shumi wa anime o miru koto desu. Douzo yoroshiku onegaishimasu.  

Perkenalkan. Nama saya Azuka Hilmi. Tolong panggil saja Azuka. Saya 15 tahun. Kelas 1 SMA. Saya tinggal di Tokyo. Hobi saya menonton anime. Senang berkenalan dengan anda.

28. kalau alamat dalam bahasa jepang menggunakan katakana apa hiragana??


=== Jawaban ===
Kalau bahasa Jepang nya "alamat", tentu saja diulis menggunakan Kanji / Hiragana.
↪Alamat; 住所 / Juusho ✔

Tetapi, jika menuliskan sebuah "alamat" lengkap, tergantung daerahbyang di tuju. Jika alamatnya terdapat di luar Jepang, diulis mrnggunakan katakana.
Jika di dlaam Jepang, kalau ada kanji nya ulis pakai kanji, kalau tidak ada kanjinya bisa di tulis menggunakan Hiragana.

Misalkan contoh "nama alamat sekolah"
Semisalkan kamu bersekolah di SMAN 1 Ambarawa penulisannya menjadi:
↪アムバラワ一の高校 (Amubarawa ichi no koukou: SMAN 1 Ambarawa ✔

❇Penjelasan:
↪アムバラワ (amubarawa): Ambarawa. Karena nama wilayah luar Jepang, harus di tulis menggunakan katakana.
↪一 (ichi): satu
↪の (no): partikel menyatakan kepemilikan
↪高校 (koukou): SMA

34. perkenalkan dirimu dalam bahasa jepang,minimal nama tempat tanggal lahir dan alamat!​


First step, Lakukan Ojigi

初めまして

Hajimemash|te

私は.....です

Watashiwa ....(isi nama) desu

インドネシアから来ました

Indonesia kara kimash|ta (jika dari Indonesia)

私は....年.....月....日に生まれました

Watashi wa (tahun lahir) nen (bulan lahir) gatsu (tanggal lahir) nichi ni umaremash|ta

.......に住んでいます

...(isi kota/kabupaten) ni sunde imasu

趣味は....ことです

shumi wa .....(isi hobi) koto desu

どうぞよろしくお願いします"

douzo yoroshiku onegaishimasu

• • • • • • • • • • • • • • • • • • • • • • • • • • • • • •

Pembahasan

Perkenalan diri dalam bahasa jepang disebut dengan Jikoshoukai (Jiko : diri/jiwa, shoukai : memperkenalkan). Dalam perkenalan diri, biasanya orang Jepang akan melakukan Ojigi yaitu tata perilaku yang dilakukan dengan membungkukan badan sebagai bentuk penghormatan.

Setelah melakukan Ojigi, bisa dilanjutkan dengan mengucapkan salam yang biasanya menggunakan kata 初めまして (Hajimemash|te) yang artinya salam kenal. Setelah itu, mereka akan memperitahy sesuatu tentang diri mereka seperti nama, tempat tanggal lahir dan lain sebagainya.

Memperkenalkan Nama

Menggunakan kalimat berikut :

私は.....です (Watashi wa ....desu). Saya adalah...(nama)私の名前は....です (Watashi no namae wa.... desu). Nama saya adalah.....私は.....と言います (Watashi wa rizu to iimasu). Saya adalah...

Memperkenalkan Kejadian Lahir/Umur

私は....年.....月....日に生まれました (Watashi wa (tahun kelahiran) nen (bulan kelahiran) gatsu (tanggal kelahiran) nichi ni umaremash|ta). Saya lahir tanggal .......(example : 17 Agustus 1945)私は...さいです (Watashi wa ... sai desu). Saya berumur....tahun

Memperkenalkan Tempat Keberadaan

......から来ました (..... kara kimash|ta). Saya dari .........に住んでいます (..... ni sunde imasu). Saya tinggal di...........で働いています(..... de hataraite imasu). Saya bekerja di ..........をしています (.... o sh|te imasu). Saya bekerja menjadi.....趣味は....ことです (shumi wa ..... koto desu). Hobi saya adalah .....

Mengucapkan Salam Akhir

どうぞよろしくお願いします. (Douzo yoroshiku onegaishimasu). Senang berkenalan dengan kamu.
