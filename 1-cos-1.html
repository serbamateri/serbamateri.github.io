29. Nilai dari ekspresi cos-1 (cos π/3)-cos (cos-1 (1/2))+tan -1(tan π/4) adalah​.


Terdapat sebuah ekspresi trigonometri:

cos⁻¹(cos ⅓π)-cos(cos⁻¹(½))+tan⁻¹(tan ¼π)

Nilai ekspresi tersebut adalah ¹⁄₁₂(7π-6).

Penjelasan dengan langkah-langkah

Diketahui: cos⁻¹(cos ⅓π)-cos(cos⁻¹(½))+tan⁻¹(tan ¼π)

Ditanya: nilai ekspresi tersebut

Jawab:

Batasan

Karena fungsi trigonometri periodik, maka fungsi inversnya memiliki nilai yang banyak untuk sebuah input angka. Pembatasannya adalah sebagai berikut:

cos⁻¹(k) = x ⇔ cos x = k, 0 ≤ x ≤ πtan⁻¹(k) = x ⇔ tan x = k, -½π ≤ x ≤ ½πNilai-nilai trigonometricos ⅓π = ½tan ¼π = 1Nilai ekspresi

cos⁻¹(cos ⅓π)-cos(cos⁻¹(½))+tan⁻¹(tan ¼π)

= cos⁻¹(½)-cos(⅓π)+tan⁻¹(1)

= ⅓π-½+¼π

= ⁴⁄₁₂π-⁶⁄₁₂+³⁄₁₂π

= ⁷⁄₁₂π-⁶⁄₁₂

= ¹⁄₁₂(7π-6)

Jadi, nilai dari ekspresi tersebut adalah ¹⁄₁₂(7π-6).
