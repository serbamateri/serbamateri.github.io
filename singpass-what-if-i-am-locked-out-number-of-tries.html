7. I think of a pair of number. If I add 11 to the first, I obtain a number which is twice the second. If I add to the second, I obtain a number which is twice the first. What are the numbers?


Pair of number itu kita misalkan x dan y.

Bila x ditambah 11, maka x sama dengan 2y
x + 11 = 2y
11 = 2y – x

Bila y ditambah 11, maka y sama dengan 2x
y + 11 = 2x
11 = 2x – y

Lalu kita gabung dua persamaan di atas
2y – x = 2x – y
3y = 3x
y = x

2y – x = 11
2(x) – x = 11
x = 11
y = 11

Jadi, nomor yang dimaksud adalah 11.

38. In a fruit stall, there are apples, bananas, blueberries and pears. The number of blueberries is 25% of the total number of fruits. The number of blueberries and bananas is the same and the number of pears is half the number of bananas. Find out the total number of fruits if there are 225 apples.


Jawab:

Penjelasan dengan langkah-langkah:

Let's break down the information provided step by step and use it to find the total number of fruits.

   We are given that there are 225 apples.

   The number of blueberries is 25% of the total number of fruits, which means 25% of the total fruits are blueberries.

   The number of blueberries and bananas is the same. Let's denote this number as "x."

   The number of pears is half the number of bananas, which means the number of pears is (1/2) * x.

Now, let's create an equation based on the information given:

   Number of blueberries + Number of bananas + Number of apples + Number of pears = Total number of fruits

We know that the number of blueberries is 25% of the total number of fruits, so it can be written as 0.25 * Total number of fruits.

We also know that the number of blueberries and bananas is the same, which is "x," so we can write it as:

Number of blueberries = Number of bananas = x

The number of pears is half the number of bananas, so:

Number of pears = (1/2) * x

Now, we can write the equation:

0.25 * Total number of fruits + x + 225 (for apples) + (1/2) * x (for pears) = Total number of fruits

Let's solve for the total number of fruits:

0.25 * Total number of fruits + x + 225 + (1/2) * x = Total number of fruits

Now, let's isolate the Total number of fruits on one side of the equation:

0.25 * Total number of fruits + x + 225 + (1/2) * x - Total number of fruits = 0

Combine like terms:

0.25 * Total number of fruits + (1/2) * x - Total number of fruits = -225

Now, let's factor out the Total number of fruits on the left side:

Total number of fruits * (0.25 - 1) + (1/2) * x = -225

Simplify:

Total number of fruits * (-0.75) + (1/2) * x = -225

Now, isolate the Total number of fruits:

Total number of fruits * (-0.75) = -225 - (1/2) * x

Divide both sides by -0.75 to find the Total number of fruits:

Total number of fruits = (-225 - (1/2) * x) / (-0.75)

Total number of fruits = (300 + 0.5x) / 0.75

Now, we need to find a value of "x" that satisfies the given conditions. We know that there are 225 apples, so the total number of fruits must be greater than or equal to 225.

Let's start by trying different values of "x" and see if we get a total number of fruits that is greater than or equal to 225:

   If x = 1 (Number of blueberries and bananas is 1):

   Total number of fruits = (300 + 0.5 * 1) / 0.75 = 400

   If x = 2 (Number of blueberries and bananas is 2):

   Total number of fruits = (300 + 0.5 * 2) / 0.75 = 500

   If x = 3 (Number of blueberries and bananas is 3):

   Total number of fruits = (300 + 0.5 * 3) / 0.75 = 600

It looks like when x = 3, we have a total number of fruits that is greater than or equal to 225 (600 is greater than 225). Therefore, the total number of fruits in the stall is 600.

So, there are 600 fruits in total in the stall, including apples, bananas, blueberries, and pears.
