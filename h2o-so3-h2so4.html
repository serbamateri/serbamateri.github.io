9. Asam Sulfat (H2SO4) dapat di produksi menurut reaksi SO3 + H2O → H2SO4 ∆Hr = XkJ jika diketahui ∆Hf SO3 = -396 kJ, ∆Hf H2O = -286 kJ dan ∆Hf H2SO4 = -814 kJ maka besarnya nilai X adalah


SO3 + H2O → H2SO4 ∆Hr = X kJ/mol
∆Hf SO3 = -396 kJ/mol
∆Hf H2O = -286 kJ/mol
∆Hf H2SO4 = -814 kJ/mol

∆Hr = [ ∆Hf produk ] - [ ∆Hf reaktan ]
X = ( ∆Hf H2SO4 ) - ( ∆Hf SO3 + ∆Hf H2O )
X = -814 - (-396 - 286)
X = - 132 kJ/mol

20. Reaksi berikut yang merupakan reaksi redoks adalah.... A. * NH3 + HCl ===> NH4Cl; B. NH4+ + OH- ===> NH3 + H2O; C. PbO + 2H+ ===> Pb2+ + H2O; D. Zn + 2H+ ===> Zn2+ + H2; E. SO3 + H2O ===> H2SO4


Jawaban:

--------------------------------------------------------------------------------------------------------------

Penjelasan:

A. NH3 + HCl ⇒ NH4Cl

biloks N pada NH3 = -3

biloks H pada NH3 = +1

biloks H pada HCl = +1

biloks Cl pada HCl = -1

biloks N pada NH4Cl = -3

biloks H pada NH4Cl = +1

biloks Cl pada NH4Cl = -1

pada reaksi ini, tidak terjadi perubahan biloks. Maka, reaksi ini bukan reaksi redoks

--------------------------------------------------------------------------------------------------------------

B. NH4⁺ + OH⁻ ⇒ NH3 + H2O

biloks N pada NH4⁺ = -3

biloks H pada NH4⁺ = +1

biloks O pada OH⁻ = -2

biloks H pada OH⁻ = +1

biloks N pada NH3 = -3

biloks H pada NH3 = +1

biloks H pada H2O = +1

biloks O pada H2O = -2

pada reaksi ini, tidak terjadi perubahan biloks. Maka, reaksi ini bukan reaksi redoks

--------------------------------------------------------------------------------------------------------------

C. PbO + 2H⁺ ⇒ Pb²⁺ + H2O

biloks Pb pada PbO = +2

biloks O pada PbO = -2

biloks H pada H⁺ = +1

biloks Pb pada Pb²⁺ = +2

biloks H pada H2O = +1

biloks O pada H2O = -2

pada reaksi ini, tidak terjadi perubahan biloks. Maka, reaksi ini bukan reaksi redoks

--------------------------------------------------------------------------------------------------------------

D. Zn + 2H⁺ ⇒ Zn²⁺ + H2

biloks Zn = 0

biloks H pada H⁺ = +1

biloks Zn pada Zn²⁺ = +2

biloks H pada H2 = 0

pada reaksi ini, terjadi perubahan biloks:

Zn dari 0 ke +2 (naik = oksidasi)

H dari +1 ke 0 (turun = reduksi)

maka, reaksi ini disebut reaksi redoks

--------------------------------------------------------------------------------------------------------------

E. SO3 + H2O ⇒ H2SO4

biloks S pada SO3 = -6

biloks O pada SO3 = -2

biloks H pada H2O = +1

biloks O pada H2O = -2

biloks H pada H2SO4 = +1

biloks S pada H2SO4 = +6

biloks O pada H2SO4 = -2

pada reaksi ini, tidak terjadi perubahan biloks. Maka, reaksi ini bukan reaksi redoks

38. Tentukan jenis reaksi berikut termasuk reaksi redoks atau bukan redoks! a. SO3 + H2O-> H2SO4 b. SnCl2 + l2 + 2HCl-> SnCl4 + 2Hl Serta caranya ​


Reaksi SO₃ + H₂O ⇒ HSO₄ bukan termasuk reaksi redoks. Hal ini karena tidak terjadi perubahan bilangan oksidasi pada reaksi SO₃ + H₂O ⇒ HSO₄. Pada reaksi SO₃ + H₂O ⇒ HSO₄ bilangan oksidasi atom S sebelum dan sesudah reaksi sama yaitu +6. Pada reaksi SO₃ + H₂O ⇒ HSO₄ bilangan oksidasi atom O sebelum dan sesudah reaksi sama yaitu -2. Pada reaksi SO₃ + H₂O ⇒ HSO₄  bilangan oksidasi atom H sebelum dan sesudah reaksi sama yaitu +1.

Pembahasan

Reaksi SnCl₂ + I₂ + 2 HCl ⇒ SnCl₄ + 2 HI adalah rekasi redoks karena pada reaksi  SnCl₂ + I₂ + 2 HCl ⇒ SnCl₄ + 2 HI terjadi rekasi reduksi dan reaksi oksidasi yaitu

∴ Bilangan oksidasi ( biloks)  I turun dari 0 menjadi -1 ( reduksi)

∴ Biloks Sn naik dari +2 menjadi + 4(oksidasi)

**************

∴ Bilangan oksidasi I pada I₂ adalah 0 karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0)

∴ Bilangan oksidasi I pada HI adalah -1

Tentukan reaksi ionisasi dari senyawa tersebut terlebih dahulu

HI ⇒  H⁺ + I⁻

Sehingga di peroleh bilangan oksidasi  dari atom H = +1 ( Karena bilang oksidasi suatu ion monoatomik adalah sesuai dengan jumlah muatannya ) dan di peroleh bilangan oksidasi  dari atom I = -1 ( Karena bilangan oksidasi senyawa halogen ( F, Cl, Br, I) pada umumnya adalah -1  )

∴ Bilangan oksidasi Sn pada SnCl₂ adalah +2.

Diperoleh dari atom  yang sudah diketahui bilangan oksidasinya pada SnCl₂ untuk memperoleh bilangan oksidasi Sn

∵Bilangan oksidasi total SnCl₂  adalah 0 ( karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) ) dan bilangan oksidasi Cl adalah -1 ( karena Bilangan oksidasi senyawa halogen ( F, Cl, Br, I) pada umumnya adalah -1   )

SnCl₂ = 0

(jumlah atom Sn x biloks atom Sn) + (jumlah atom Cl x biloks atom Cl) = 0

(1 x biloks Sn ) + (2x (-1)) = 0

Biloks Sn - 2 = 0

Biloks Sn = 0+2

Biloks Sn = +2

∴ Bilangan oksidasi Sn pada SnCl₄ adalah +4.

Diperoleh dari atom  yang sudah diketahui bilangan oksidasinya pada SnCl₄ untuk memperoleh bilangan oksidasi Sn

∵Bilangan oksidasi total SnCl₄  adalah 0 ( karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) ) dan bilangan oksidasi Cl adalah -1 ( karena Bilangan oksidasi senyawa halogen ( F, Cl, Br, I) pada umumnya adalah -1   )

SnCl₂ = 0

(jumlah atom Sn x biloks atom Sn) + (jumlah atom Cl x biloks atom Cl) = 0

(1 x biloks Sn ) + (4x (-1)) = 0

Biloks Sn - 4 = 0

Biloks Sn = 0+4

Biloks Sn = +4
