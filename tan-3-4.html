[1 - tan A] / [1 + tan A] = [4 + 4tan A] / [1 - tan A)]

[1 - tan A]² = [1 + tan A] x [4 + 4tan A]

tan²A - 2tan A + 1 = 4tan²A + 8tan A + 4

Jika tan x = tan a maka x=a+k·180º.

Berdasarkan rumus di atas maka

tan (A + B) = (tan A + tan B)/(1- tan A·tan B)

tan (A + B) = (4/3 + 7)/(1- 4/3·7)

tan (A + B) = (4/3 + 21/3)/(1- 28/3)

tan (A + B) = (25/3)/(3/3- 28/3)

tan (A + B) = (25/3)/(-25/3)

tan (A + B) = (25/3)·(-3/25)

tan (A + B) =-1

tan (A + B) = tan (-45º)

A + B = -45º+k·180º

Untuk k=1 maka A + B = -45º+1·180º=135º.

Untuk k=2 maka A + B = -45º+2·180º= -45º+360º=315º.

Jadi, salah satu hasil A + B adalah 135º.

36. jika tan A=1/3 maka nilai tan(pi/4-A)/tan(pi/4+A) adalah


Jika tan A = 1/3 maka nilai tan (pi/4 – A) / tan (pi/4 + A) adalah ¼. Hasil tersebut diperoleh dengan menggunakan rumus jumlah dan selisih sudut pada tangen.  

Rumus jumlah dan selisih sudut trigonometri

sin (A + B) = sin A cos B + cos A sin B sin (A – B) = sin A cos B – cos A sin B cos (A + B) = cos A cos B – sin A sin B cos (A – B) = cos A cos B + sin A sin B tan (A + B) = [tex]\frac{tan \: A + tan \: B}{1 - tan \: A tan \: B}[/tex] tan (A – B) = [tex]\frac{tan \: A - tan \: B}{1 + tan \: A tan \: B}[/tex]
Pembahasan

Diketahui

tan A = 1/3


Ditanyakan

Nilai [tex]\frac{tan(\frac{\pi}{4} - A)}{ tan(\frac{\pi}{4} + A)}[/tex] = ...?


Jawab


[tex]tan(\frac{\pi}{4} - A)[/tex]

= [tex]\frac{tan \: \frac{\pi}{4} - tan \: A}{1 + tan \: \frac{\pi}{4} tan \: A}[/tex]

= [tex]\frac{1 - \frac{1}{3}}{1 + 1. \frac{1}{3}}[/tex]

= [tex]\frac{\frac{3}{3} - \frac{1}{3}}{\frac{3}{3} + \frac{1}{3}}[/tex]

= [tex]\frac{\frac{2}{3}}{\frac{4}{3}}[/tex]

= [tex]\frac{2}{4}[/tex]

= [tex]\frac{1}{2}[/tex]


[tex]tan(\frac{\pi}{4} + A)[/tex]

= [tex]\frac{tan \: \frac{\pi}{4} + tan \: A}{1 - tan \: \frac{\pi}{4} tan \: A}[/tex]

= [tex]\frac{1 + \frac{1}{3}}{1 - 1. \frac{1}{3}}[/tex]

= [tex]\frac{\frac{3}{3} + \frac{1}{3}}{\frac{3}{3} - \frac{1}{3}}[/tex]

= [tex]\frac{\frac{4}{3}}{\frac{2}{3}}[/tex]

= [tex]\frac{4}{2}[/tex]

= 2


Jadi


Nilai [tex]\frac{tan(\frac{\pi}{4} - A)}{ tan(\frac{\pi}{4} + A)}[/tex]

= [tex]\frac{\frac{1}{2}}{2}[/tex]

= [tex]\frac{1}{2} \div 2[/tex]

= [tex]\frac{1}{2} \times \frac{1}{2} [/tex]

= [tex]\frac{1}{4}[/tex]
