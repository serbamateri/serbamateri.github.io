1. Hitunglah sin (a-b) jika sin a = 3/5 dan sin b = 3/5


jwb

sin a = 3/5

sin b = 3/5

cos a = ±√1 - sin^2 a (pitagoras)

cos b = ±√1 - sin^2 b

a = √1 - sin^2 a

= √1 - 3/5^2

= √1 - 9/25

= √16/25

= 4/5

b = √1 - sin^2 b

= √1 - 3/5^2

= √1 - 9/25

= √16/25

= 4/5

sin(a - b) = sin a cos b - cos a sin b

= (3/5)(4/5) - (4/5)(3/5)

= 12/25 - 12/25

= 0

sin(a - b) = 0

Jawaban:

Untuk menyelesaikan soal ini, kita perlu menggunakan rumus trigonometri untuk sin (a-b), yaitu:

sin (a-b) = sin a cos b - cos a sin b

Diketahui sin a = 3/5 dan sin b = 3/5, serta cos a dan cos b tidak diketahui. Namun, kita dapat menggunakan identitas Pythagoras untuk mencari nilai cos a dan cos b, yaitu:

cos^2 a + sin^2 a = 1

cos^2 b + sin^2 b = 1

Maka, kita bisa mencari nilai cos a dan cos b sebagai berikut:

cos a = √(1 - sin^2 a) = √(1 - (3/5)^2) = √(1 - 9/25) = √(16/25) = 4/5

cos b = √(1 - sin^2 b) = √(1 - (3/5)^2) = √(1 - 9/25) = √(16/25) = 4/5

Selanjutnya, kita bisa substitusikan nilai sin a, cos a, sin b, dan cos b ke dalam rumus sin (a-b), sehingga:

sin (a-b) = sin a cos b - cos a sin b

= (3/5 x 4/5) - (4/5 x 3/5)

= 12/25 - 12/25

= 0

Jadi, nilai sin (a-b) adalah 0.
