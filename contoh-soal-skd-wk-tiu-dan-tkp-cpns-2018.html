Penjelasan dengan langkah-langkah:

jarak = 500 km

kecepatan = x km/jam

waktu = w + 3/4

jarak = kecepatan x waktu

500 = x (w + 3/4)

500 = xw + 3/4x ........ persamaan I

jarak = 400

kecepatan = x km/jam

w = w - 1/2

400 = x (w-1/2)

400 = xw -1/2x .............. persamaan II

persamaan I - Persamaan II (metode eliminasi xw)

100 = 3/4x - (-1/2x)

100 = 3/4x + 1/2x

100 = (3+2) /4 x

100 = 5/4 x

x = 100 x 4/5

x = 80 km/jam (D)

s1 = 500 km

w1 = t + 3/4 jam

s2 = 400 km

w2 = t - 1/2 jam

s1/w1 = s2/w2

500/(t + 3/4) = 400/(t - 1/2)

500(t - 1/2) = 400(t + 3/4)

500t - 250 = 400t + 300

500t - 400t = 300 + 250

100t = 550

t = 550/100

t = 5,5 jam

v1 = s1/w1

v1 = s1/(t + 3/4)

v1 = 500/(5,5 + 3/4)

v1 = 500/(5,5 + 0,75)

v1 = 500/6,25

v1 = 80

jadi x adalah 80 km/jam (D)

Tahapan-tahapan tersebut merupakan tahapan dari Manajemen Pemeliharaan.

Untuk menghitung matriks 2A + B - C, pertama-tama kita perlu mengalikan matriks A dengan 2. Kemudian kita jumlahkan matriks hasil perkalian dengan matriks B dan kurangi dengan matriks C. Langkah-langkahnya adalah sebagai berikut:

2A =

2 x /-1 3/ = /-2 6/

/3 6/

2A + B =

/-2 6/ + /-1 -3/ = /-3 3/

/3 6/ /3 6/

2A + B - C =

/-3 3/ - /-1 3/ = /-2 0/

/3 6/ /3 3/

Jadi, matriks 2A + B - C adalah:

/-2 0/

Untuk menyelesaikan sistem persamaan linear, kita dapat menggunakan metode eliminasi Gauss-Jordan. Pertama-tama, kita ubah sistem persamaan tersebut ke dalam bentuk matriks augmented dan kemudian kita lakukan operasi baris-elementer untuk mendapatkan matriks segitiga atas. Langkah-langkahnya adalah sebagai berikut:

/i> 4x + 3y = 5

(ii) 6x - 2y = 5

Bentuk matriks augmented:

/4 3 | 5/

/6 -2 | 5/

Kita kurangkan 1,5 kali baris pertama dari baris kedua:

/4 3 | 5/

/0 -11/2 | -5/2/

Kita bagi baris kedua dengan -11/2:

/4 3 | 5/

/0 1 | 5/11/

Kita kurangkan 3 kali baris kedua dari baris pertama:

/4 0 | -10/11/

/0 1 | 5/11/

Kita dapatkan nilai x dan y sebagai berikut:

x = -10/11

y = 5/11

Jadi, nilai x dan y adalah -10/11 dan 5/11. Oleh karena itu, jawaban yang benar adalah A.
