2. Jelaskan kelebihan kunci pas-ring dibanding kunci pas


Salah satu alat yang bisa digunakan ketika berurusan dengan mesin yang ada pada kendaraan bermotor adalah kunci pas ring dan kunci pas. Ketimbang kunci pas, berikut adalah beberapa kelebihan dari kunci pas ring:

Lebih mudah digunakan: kunci pas ring dibuat menyatu dalam satu tuas berukuran sama yang membuat kita bisa menggunakan kunci tersebut untuk memutar kepala baut sambil menjaga kepala baut untuk tidak bergeser atau slek.Digunakan pada sudut sempit: kunci pas ring bisa digunakan untuk menguatkan atau melonggarkan baut pada sudut putar yang lebih sempit ketimbang kunci pas biasa karena sudut minimal dari kunci pas adalah 90 derajat.Area memegang yang lebih luas: kunci pas memiliki area pegangan terhadap kepala baut atau mur yang lebih luas sehingga permukaan kunci ring akan mengikat dan mencengkeram bagian baut dengan lebih baik ketimbang kunci pas.

Pembahasan

Kunci pas adalah jenis kunci yang memiliki ujung saling terbuka sesuai dengan ukuran kunci pada kedua ujungnya sehingga ukuran dari kunci pas harus sesuai dengan baut yang akan dikerjakan. Sedangkan, kunci pas ring memiliki bentuk ring atau lingkaran pada salah satu ujungnya yang bisa digunakan untuk mencengkeram kepala baut dengan lebih baik ketimbang kunci pas.

22. Jelaskan penggunaan kunci ring dan kunci pas


Jawaban:

Fungsi kunci ring adalah untuk mengencangkan dan mengendurkan mur maupun baut yang ada di dalam komponen kendaraan bermotor. Namun, jenis baut dan mur yang bisa diatur posisinya menggunakan kunci ring hanyalah tipe berbentuk heksagonal atau segi enam. Baut segi enam ini banyak digunakan dalam dunia otomotif. Penjelasan:

Mudah saja apabila mau memakai fungsi kunci ring untuk mengendurkan maupun mengencangkan posisi mur dan baut mobil. Tinggal memutarnya ke arah kiri untuk mengencangkan posisi baut dan mur. Lalu, putar ke kanan untuk mengendurkan mur dan baut.  

Akan tetapi, sebelum itu pastikan dulu kalau ukuran mur atau baut sesuai dengan ukuran kunci ring. Caranya, coba masukkan mur atau baut ke kunci ring untuk melihat apakan ukurannya sesuai. Kalau sudah, letakkan dan pasang kunci tepat pada baut. Posisikan tegak lurus agar tidak meleset. Baru kemudian putar untuk mengencangkan maupun mengendurkan.

Besaran merupakan suatu pengukuran dalam fisika dan satuan adalah unit yang terukurnya .

I.Besaran : Sesuatu yang dapat diukur, memiliki nilai dan satuan  

Pembagian Besaran  

1.Berdasar Satuan  

a.Besaran Pokok : Besaran yang satuan sudah terdefinisikan dan berdiri sendiri  

Contoh :  

( J I Wa Ku S M P )  

Jumlah zat ( mol ), Intensitas cahaya ( Cd ), Waktu ( s ), Kuat arus ( A ), Suhu ( K ), Massa ( Kg ) dan Panjang ( m )  

b.Besaran Turunan : Besaran yang satuannya diturunkan dari satuan besaran pokok  

Contoh :  

Luas ( m2 ), Volume ( m3 ), Massa jenis ( Kg/m3 ), Kecepatan ( m/s ), Percepatan ( m/s2 ), Gaya ( N = kgm/s2 ), Energi ( J = Nm = Kgm2/s2 ), Daya ( Watt = J/s = Kgm2/s3 ), Tekanan ( N/m2 = Kg/ms2 ), Momentum ( Kgm/s ), Impuls ( Kgm/s ), Usaha (J)  

2.Berdasar Arah  

a.Besaran Skalar : Besaran yang hanya memiliki nilai saja  

Contoh : Semua besaran pokok termasuk besaran skalar, Luas, Volume, Kelajuan, Perlajuan, Massa Jenis, Energi, Daya dsb  

b.Besaran Vektor : Besaran yang memiliki nilai dan arah  

Contoh : Perpindahan, Kecepatan, Percepatan, Gaya, Momentum dsb  

II.Dimensi  

Merupakan gambaran bagaimana besaran tersebut tersusun atas besaran-besaran pokok.  

Fungsi dimensi : untuk menguji benar / setara tidaknya suatu persamaan ( Persamaan yang benar secara dimensi belum tentu benar secara fisis )  

Dimensi dari besaran – besaran pokok :  

Jumlah zat [ N ], Intensitas cahaya [ J ], Waktu [ T ], Kuat arus [ I ], Suhu [ θ ], Massa [ M ] dan Panjang [ L ]

Pembahasan  

Perhatikan gambar, semuanya dapat diukur dengan jangka sorong

Tang, obeng, pahat => rahang bawah

mur, kunci pas, kunci ring  => rahang atas

Makhluk ekonomi adalah suatu makhluk yang selalu berusaha untuk memenuhi kebutuhannya dengan memanfaatkan berbagai sumber daya alam yang tersedia. Berdasarkan soal, dapat disimpulkan bahwa alat yang dalam penggunaannya hanya mengandalkan tenaga manusia adalah hand tool (E). Akan tetapi, meskipun dengan mesin tetap membutuhkan kontrol dari manusia.

1 set kunci L + 2 set kunci ring pas = 62.000 I x1

1 set kunci L + 2 set kunci ring pas = 62.000

Eliminasi

= 3 set kunci L = 18.000

= 1 set kunci L = 6000

Substitusi
