PEMBAHASAN :

Hukum tajwid adalah ilmu yang mempelajari bagaimana cara membaca atau mengucapkan huruf-huruf yang terdapat dalam kitab suci Al-Qur'an dengan benar

Tajwidnya sendiri ada dua, yaitu:

Syafawi ‘Amali, yaitu bacaan Al-Quran yang bagus yang diambil dari orang yang ahli dalam membaca Al-QuranNadzory ‘Ilmi, yaitu suatu ilmu yang diajarkan secara turun-temurun menurut kaidah yang diletakkan oleh para ulama.

Idhar :

Memiliki arti dijelaskan atau memperjelas, Menurut istilah Idhar adalah apabila nun mati atau sukun ( نْ ) dan tanwin bertemu dengan huruf idhar yaitu hamzah (ء), kha (ح ), kho (خ), 'ain (ع), gho (غ), dan ha' (هـ )

Ikhfa' :

Memiliki arti disamarkan atau menyamarkan, Menurut istilah Ikhfa' adalah apabila nun mati atau sukun ( نْ ) dan tanwin bertemu dengan huruf ikhfa' yaitu fa' ( ف ), zha ( ظ ), tha ( ط ), tsa' ( ث ), dhad ( ض ), shad ( ص ), syin ( ش ), sin ( س ), za' ( ز ), kaf ( ك ), qaf ( ق ), dzal ( ذ ), dal ( د ), ta' ( ت ), dan jim ( ج )

Idgham bighunnah :

Memiliki arti meleburkan dengan dengung, Menurut istilah Idgham bighunnah adalah apabila nun mati atau sukun ( نْ ) dan tanwin bertemu dengan huruf Idgham bighunnah yaitu nun ( ن ), ya' ( ي ), mim ( م ), dan wau ( و )

Idgham bilaghunnah :

Memiliki arti meleburkan tanpa dengung, Menurut istilah Idgham bilaghunnah adalah apabila nun mati atau sukun ( نْ ) dan tanwin bertemu dengan huruf Idgham bilaghunnah yaitu lam ( ل ), dan Ro ( ر )

Iqlab :

Memiliki arti membalikkan atau mengganti, Menurut istilah Iqlab adalah apabila nun mati atau sukun ( نْ ) dan tanwin bertemu dengan huruf Iqlab yaitu ba' ( ب )

Al - Qamariyah adalah ( ال ) yang

dirangkai dengan kata isim yang

diawali dengan salah satu dari huruf huruf qamariyah, Qamariyah artinya seperti Bulan satu

Jumlah huruf qamariyah ada 14 huruf, yaitu huruf-huruf hijaiyah :

( ا ب غ ح ج ك و خ ف ع ق ي م ه )

Al - Syamsiyah adalah ( ال ) atau alif lam yang dirangkai dengan kata isim yang diawali dengan salah satu dari huruf-huruf syamsiyah, Syamsiah artinya seperti matahari

Jumlah huruf syamsiyah ada 14 huruf, yaitu huruf-huruf hijaiyah :

( ن د س ظ ز ش ل ط ث ص ر ت ض ذ )
