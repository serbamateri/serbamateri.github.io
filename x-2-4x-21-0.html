Penjelasan dengan langkah-langkah:

f(x) = x² - 4x - 21 -> a = 1, b = -4, c = -21

Titik balik/puncak (xp, yp)

xp = -b/2a

xp = -(-4) / 2(1)

xp = 4/2

xp = 2

yp = (b² - 4ac) / -4a

yp = ((-4)² - 4(1)(-21)) / -4(1)

yp = (16 + 84) / -4

yp = 100/-4

yp = -25

Jadi, titik balik pada fungsi adalah (2, -25)
