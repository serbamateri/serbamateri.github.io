4. Buatlah program untuk menghitung gaji pegawai Input : nama, golongan Proses : · Jika golongan A maka gaji pokok =Rp 1.500.000, tunjangan =5% * gaji pokok .jika gol.B maka gaji pokok = Rp.1.700.000,tunjangan = 10%* gaji pokok .jika gol.C maka gaji pokok = Rp.2.000.000, tunjangan = 15%*gaji pokok Output : Nama,gaji pokok,tunjangan serta gaji bersih Petunjuk : gunakan array dan petualangan


program untuk menghitung gaji pegawai, program ditulis dalam bahasa C

#include <stdio.h>

int main()

{

    char nama[50];

    char golongan;

    int gaji[3] = { 1500000, 1700000, 2000000 };

    int persentunjangan[3] = { 5, 10, 15 };

    int gajibersih, tunjangan;

    printf("Masukan nama Anda : ");

    scanf("%s", nama);

    printf("Masukan golongan (A, B, atau C) : ");

    scanf("%c", &golongan);

    golongan = golongan - 'A';

    tunjangan = gaji[ golongan ] * persentunjangan[ golongan ] / 100;

    gajibersih = gaji[ golongan ] + tunjangan;

    printf("Nama : %s\n", nama);

    printf("Gaji pokok : %i\n", gaji[ golongan ]);

    printf("Tunjangan : %i\n", tunjangan);

    printf("Gaji bersih : %i\n", gajibersih);

    return 0;

}

Pembahasan

Ada bagian yang aneh pada soal, "gunakan array dan petualangan". Array adalah bentuk variabel dengan alokasi memory untuk banyak nilai. Yang saya bingung adalah istilah "petualangan" disini. Setahu saya tidak ada istilah "petualangan" dalam bahasa pemrograman, mohon dikoreksi jika memang terjadi typo.

Pada solusi pemrograman yang dibuat, kita memanfaatkan golongan untuk menjadi index dari Array.

Pada potongan code berikut golongan = golongan - 'A';

Berfungsi untuk mengubah karakter A menjadi nilai 0. Perlu diketahui karakter adalah sebuah huruf yang memiliki nilai angka pada komputer, atau ASCII. Sehingga bisa dioperasikan menggunkana operator aritmatika seperti penjumlahan pengurangan, dan lainnya.

Maka jika golongan adalah A dan dikurangi 'A' maka hasilnya adalah 0

jika golongan adalah B dan dikurangi 'A' maka hasilnya adalah 1

jika golongan adalah C dan dikurangi 'A' maka hasilnya adalah 2

Nilai ini bisa kita gunakan sebagai index dalam array gaji yang sudah didefinisikan sebelumnya.

Sehingga kita bisa akses secara dinamis array gaji menggunakan golongan sebagi indexnya.

11. faktor penghambat dan penunjang adanya pancasila di perguruan tinggi


Faktornya karena pemuda pemuda zaman sekarang menganggap pancasila sebagai suatu simbol negara dan mulai melupakan nilai nilai yang terkandung didalamnya. Padahal pancasila yang menjadi dasar negara dan sumber dari segala hukum dan perundang undangan adalah nafas bagi eksitensi bangsa Indonesia.

Pembahasan

Pancasila berasal dari dua kata yakni panca dan sila. Panca yang memiliki arti lima sedangkan untuk Sila yang berarti asas atau prinsip. Jadi kata Pancasila adalah lima prinsip yang mempunyai kedudukan yang terdapat di Indonesia. Lalu sebagai  Identitas adalah suatu nama atau label yang sudah  melekat pada diri seseorang dan untuk menggambarkan jati diri dari orang tersebut. 12. Kartikasari, s.H. Seorang golongan pns III/b memiliki gaji pokok 3.500.000,tunjangan istri 10% tunjangan anak 2%per anak ,tunjanan fungaional 2.000.000,iuran pengsiun 4,75%.Status kawin denan 2 anak jumlah penghasilan neto kartikasari per bulan


PENGHASILAN NETTO

Penghasilan bruto (gaji pokok + tunjangan-tunjangan) yang sudah dikurangi dengan biaya-biaya (biaya jabatan, iuran pensiun, iuran BPJS, iuran asuransi, dan lain-lain)

________________________________

Langkah-Langkahnya

1. Mencari gaji pokok

2. Mencari tunjangan istri

3. Mencari tunjangan 2 anak

4. Mencari tunjangan fungsional

5. Mencari iuran pensiun

6. Mencari biaya jabatan

7. Mencari penghasilan netto

GAJI POKOK

= 3.500.000

TUNJANGAN ISTRI

= 10% x gaji pokok

= 10% x 3.500.000

= 350.000

TUNJANGAN 2 ANAK

= 2 x (2% x gaji pokok)

= 2 x (2% x 3.500.000)

= 2 x 70.000

= 140.000

TUNJANGAN FUNGSIONAL

= 2.000.000

IURAN PENSIUN

= 4,75% x gaji pokok

= 4,75% x 3.500.000

= 166.250

BIAYA JABATAN

= 5% x gaji pokok

= 5% x 3.500.000

= 175.000

PENGHASILAN NETTO

= (gaji pokok + tunjangan istri + tunjangan 2 anak + tunjangan fungsional) - (iuran pensiun + biaya jabatan)

= (3.500.000 + 350.000 + 140.000 + 2.000.000) - (166.250 + 175.000)

= 5.990.000 - 341.250

= Rp5.648.750

Jadi, penghasilan netto Kartika Sari per bulannya adalah Rp5.648.000

________________________________

BEA JABATAN adalah biaya yang dikenakan kepada wajib pajak yang memperoleh jabatan tertentu di suatu instansi, baik swasta maupun pemerintahan. Jumlahnya yakni 5% dari gaji pokok. Dengan batas maksimal pengurangan yakni Rp500.000 per bulan dan Rp6.000.000 per tahunnya

38. Pemerintah daerah membutuhkan sumber pendapatan untuk tunjangan profesi guru (TPG) yang berasal dari...


Pemerintah daerah membutuhkan sumber pendapatan untuk tunjangan profesi guru (TPG) yang berasal dari APBN. TPG atau Tunjangan Profesi Guru ini ditrasfer pemerintah melalui APBN ke APBD melaui mekanisme dana transfer daerah, serta melalui APBN ke Daftar Isian Pelaksanaan Anggaran (DIPA) Kemendikbud.

Pembahasan

APBN atau Anggaran Pendapatn dan Belanja Negara merupakan daftar yang berisi rencana pemerintah dalam mengelola keuangan negara dengan membuat daftar-daftar prioritas anggaran. seperti fungsi distribusi dan alokasi yang dimiliki APBN, untuk menunjang profesi guru, pemerintah pusat membuat anggaran yang kemudian ditrasfer ke APBD untuk disalurkan kepada guru-guru di daerah yang memiliki hak atas tunjangan tersebut.

Fungsi distribusi ini diharapkan dapat memberikan dampak positif bagi perekonomian seperti pemerataan pendapatan antar daerah, sehingga tidak ada lagi kesenjangan pendapatan.

Guru merupakan profesi yang sangat penting bagi kecerdasan bangsa, karena guru berada di lini paling depan dunia pendidikan yang dapat mencetak sumber daya manusia yang berkualitas.
