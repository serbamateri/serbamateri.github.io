9. sin x = sin [tex] \frac{1}{6} \pi[/tex], 0 [tex] \leqslant \times \leqslant 2\pi[/tex]​


Trigonometri

Jawaban

Himpunan penyelesaian persamaan sin x = sin π/6 adalah [π/6, 5π/6)

Pembahasan

Definisi

Dalam sebuah segitiga siku-siku ABC berlaku:

sin x = sisi depan / sisi miring

cos x = sisi samping / sisi miring

tan x = sisi depan / sisi samping

Persamaan Trigonometri

• sin x = sin a, maka:

x1 = a + k.360° atau x2 = (180 - a)° + k.360°

• cos x = cos a, maka:

x1.2 = ± a + k. 360°

• tan x = tan a, maka:

x = a + k. 180°

Penyelesaian:

sin x = sin π/6

x = π/6

0 < x < 2π

X = a + k.2π atau X = (π - a) + k.2π

• untuk k = 0

x = π/6 + 0. 2π

x = π/6

atau

x = (π - π/6) + 0. 2π

x = 5π/6

Himpunan penyelesaian => {π/6, 5π/6}

37. Sin (x + pi/2) = 0, 6 . Tentukan nilai dari 2sin x + sin(x + pi/2) + cos(pi - x)​


Jawaban:

Mari kita selesaikan langkah demi langkah.

1. Pertama, kita tahu bahwa \(\sin(x + \frac{\pi}{2}) = 0.6\). Itu berarti \(\frac{\pi}{2}\) adalah sudut yang menyebabkan \(\sin\) bernilai 0.6. Dalam istilah matematika, kita dapat menuliskan \(\sin(\frac{\pi}{2}) = 0.6\).

2. Sekarang kita ingin menemukan nilai \(\sin(x)\). Kita tahu bahwa \(\sin(x + \frac{\pi}{2}) = \sin(x)\cos(\frac{\pi}{2}) + \cos(x)\sin(\frac{\pi}{2})\), tetapi \(\cos(\frac{\pi}{2}) = 0\) dan \(\sin(\frac{\pi}{2}) = 1\), sehingga \(\sin(x + \frac{\pi}{2}) = \sin(x)\).

3. Oleh karena itu, \(\sin(x) = 0.6\).

4. Sekarang kita ingin menemukan nilai \(\cos(x)\). Kita tahu bahwa \(\sin^2(x) + \cos^2(x) = 1\), sehingga \(\cos^2(x) = 1 - \sin^2(x) = 1 - 0.6^2\).

5. \(\cos^2(x) = 1 - 0.36\).

6. \(\cos^2(x) = 0.64\).

7. \(\cos(x) = \pm \sqrt{0.64}\).

8. \(\cos(x) = \pm 0.8\).

Sekarang kita memiliki nilai \(\sin(x) = 0.6\) dan \(\cos(x) = \pm 0.8\).

9. Sekarang mari kita hitung \(2\sin(x) + \sin(x + \frac{\pi}{2}) + \cos(\pi - x)\):

\(2\sin(x) + \sin(x + \frac{\pi}{2}) + \cos(\pi - x) = 2(0.6) + 0.6 + \cos(\pi - x)\).

10. \(\cos(\pi - x) = -\cos(x)\). Oleh karena itu, kita dapat menggantikannya dengan -0.8:

\(2(0.6) + 0.6 - 0.8 = 1.2 + 0.6 - 0.8 = 1.2 - 0.8 = 0.4\).

Jadi, nilai dari \(2\sin(x) + \sin(x + \frac{\pi}{2}) + \cos(\pi - x)\) adalah 0.4
