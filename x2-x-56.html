13. Tanpa menggunakan media selesaikanlah persamaan kuadrat berikut!1. x2 - 36x = 02. x2 - x - 56 = 03. x2 + 4x - 45 = 04. x2 - 25 = 0​


Jawab:

Gunakan kaidah mengalikan apapun dengan 0 hasilnya pasti 0.

Contoh: 2020 dikali 0 hasilnya 0. 0 dikali 0 hasilnya 0 juga.

1. x(x - 36) = 0

Maka jika x = 0, memenuhi. Jika x - 36 = 0, x = 36, memenuhi juga. Maka solusinya ada 2, x = 0 atau 36.

2. Faktorkan menjadi (x - 8)(x + 7) = 0.

x - 8 = 0, x = 8 atau x + 7 = 0, x = -7. Maka jawabannya, x = -7 atau 8.

3. Faktorkan menjadi (x + 9)(x - 5) = 0.

x + 9 = 0, x = -9 atau x - 5 = 0, x = 5. Maka jawabannya, x = -9 atau 5.

4. Faktorkan menjadi (x + 5)(x - 5) = 0.

x + 5 = 0, x = -5 atau x - 5 = 0, x = 5. Maka jawabannya, x = -5 atau 5.

Catatan tambahan:

Contoh soal nomor 1, 0 dan 36 disebut sebagai akar-akar persamaannya atau solusi dari persamaan tersebut. Akar-akar maksudnya jika dimasukkan, hasilnya akan memenuhi. Coba masukkan 1, kita akan memperoleh -35 = 0, tentu tidak memenuhi, 'kan?

Untuk soal no 2 dan 3, pemfaktoran dilakukan dengan dasar, jika a dan b akar-akar persamaan, maka

(x - a)(x - b) = 0

Jabarkan menjadi

x² - ax - bx + ab = 0

x² - (a + b)x + ab = 0

(Ingat, karena masuk ke dalam kurung dan masih minus, tandanya sama.)

Maka cari yang jumlahnya -(a + b) dan hasil kalinya ab. Ini namanya pemfaktoran.

Untuk soal no 4 kaidahnya sama dengan no 2 dan 3, namun ada pemfaktoran bentuk istimewa yang sebaiknya dihafalkan, yakni a² - b² = (a + b)(a - b). Soal seperti ini sering sekali keluar, baik pada ulangan, maupun pada olimpiade.

29. Jika massa FeSO4 dan X2(SO4)3 sama, maka perbandingan massa Fe dan massa X adalah... (ArS= 32, Fe= 56, X= 27) A. 56:27 B. 56:24 C. 171:152 D. 342:270 E. 342:560


Jika massa FeSO₄ dan X₂(SO₄)₃ adalah sama maka perbandingan massa Fe dan massa X adalah 368 : 158 atau 184 : 79

Pembahasan

[tex]\boxed{\boxed{\bold{\pink{Stoikiometri}}}}[/tex]  

Dalam bahasa Yunani, kata stoicheion berarti unsur sedangkan stoikiometri (stoichiometry) berarti mengukur unsur meliputi massa atom dan massa rumus,kadar, konsentrasi, rumus empiris, rumus molekul, persamaan kimia serta semua yang berhubungan dengan reaksi kimia. Dalam stoikiometri, satuan yang digunakan untuk menyatakan jumlah partikel dalam zat dinamakan mol.  

   

[tex]\boxed{\boxed{\bold{\green{Diket}}}}[/tex]

massa FeSO₄ = massa X₂(SO₄)₃

Ar Fe = 56

Ar S = 32

Ar O = 16

Ar X = 27

 

[tex]\boxed{\boxed{\bold{\purple{Ditanya}}}}[/tex]  

Perbandingan massa Fe dan massa X

[tex]\boxed{\boxed{\bold{\blue{Jawab}}}}[/tex]  

Menentukan Mr FeSO₄

Mr FeSO₄ = Ar Fe + Ar S + 4. Ar O

Mr FeSO₄ = 56 + 32 + 4. 16

Mr FeSO₄ = 152

--------------------------------------------------------

Menentukan Mr X₂(SO₄)₃

Mr X₂(SO₄)₃ = 2. Ar X + 3. Ar S + 12. Ar O

Mr X₂(SO₄)₃ = 2. 27 + 3. 32 + 12. 16

Mr X₂(SO₄)₃ = 342

--------------------------------------------------------  

Menentukan massa Fe dalam FeSO₄

massa FeSO₄ = massa X₂(SO₄)₃

34. jika massa sulfur dalam feso4 dan x2(so4)3 adalah sama maka perbandingan massa fe dan massa x adalah ( Ar S = 32, Fe = 56, X = 28 )


Jika massa sulfur dalam FeSO₄ dan X₂(SO₄)₃ adalah sama maka perbandingan massa fe dan massa x adalah 3  : 1

Pembahasan

[tex]\boxed{\boxed{\bold{Stoikiometri}}}[/tex]  

Dalam bahasa Yunani, kata stoicheion berarti unsur sedangkan stoikiometri (stoichiometry) berarti mengukur unsur meliputi massa atom dan massa rumus,kadar, konsentrasi, rumus empiris, rumus molekul, persamaan kimia serta semua yang berhubungan dengan reaksi kimia. Dalam stoikiometri, satuan yang digunakan untuk menyatakan jumlah partikel dalam zat dinamakan mol.  

[tex]\boxed{\boxed{\bold{Diket}}}[/tex]  

massa S dalam FeSO₄ = massa S dalam X₂(SO₄)₃

Ar Fe = 56

Ar S = 32

Ar O = 16

Ar X = 28

[tex]\boxed{\boxed{\bold{Ditanya}}}[/tex]  

Perbandingan massa Fe dan massa X

[tex]\boxed{\boxed{\bold{Jawab}}}[/tex]  

Menentukan Mr FeSO₄

Mr FeSO₄ = Ar Fe + Ar S + 4. Ar O

Mr FeSO₄ = 56 + 32 + 4. 16

Mr FeSO₄ = 152

--------------------------------------------------------

Menentukan Mr X₂(SO₄)₃

Mr X₂(SO₄)₃ = 2. Ar X + 3. Ar S + 12. Ar O

Mr X₂(SO₄)₃ = 2. 28 + 3. 32 + 12. 16

Mr X₂(SO₄)₃ = 344

--------------------------------------------------------

[tex]\boxed{\boxed{\bold{massa~A~dalam~y~gram~A_p_B_q~=~\frac{p~x~Ar~A}{Mr~ApBq}~x~y~gram~(massa~A_pB_q}}}[/tex]

--------------------------------------------------------

Menentukan massa FeSO₄

massa S dalam FeSO₄ = massa S dalam X₂(SO₄)₃

kita misalkan massa S dalam FeSO₄ = massa S dalam X₂(SO₄)₃ = A gram

massa S = [tex]\frac{1~x~Ar~S}{Mr~FeSO_4}~x~massa~FeSO_4[/tex]

A gram = [tex]\frac{1~x~32}{152}~x~massa~FeSO_4[/tex]

massa FeSO₄ = [tex]\frac{152A}{32}[/tex]

massa FeSO₄ = 4,75A gram

--------------------------------------------------------

Menentukan massa Fe

massa Fe = [tex]\frac{1~x~Ar~Fe}{Mr~FeSO_4}~x~massa~FeSO_4[/tex]

massa Fe = [tex]\frac{1~x~56}{152}~x~4,75A[/tex]

massa Fe = 1,75 A gram

--------------------------------------------------------

Menentukan massa X₂(SO₄)₃

massa S dalam FeSO₄ = massa S dalam X₂(SO₄)₃

kita misalkan massa S dalam FeSO₄ = massa S dalam X₂(SO₄)₃ = A gram

massa S = [tex]\frac{3~x~Ar~S}{Mr~X_2(SO_4)_3}~x~massa~X_2(SO_4)_3[/tex]

A gram = [tex]\frac{3~x~32}{344}~x~massa~X_2(SO_4)_3[/tex]

massa X₂(SO₄)₃ = [tex]\frac{344A}{96}[/tex]

massa X₂(SO₄)₃ = 3,583A gram

massa X = [tex]\frac{2~x~28}{344}~x~3,583A[/tex]

massa X = 0,58327906976744186046511627906977A gram

massa X = 0,583A gram
