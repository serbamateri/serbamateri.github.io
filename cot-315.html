Cot (360 - a)° = - Cot a° » Kuadran IV

Cot a° = 1/Tan a°

- Cot 45° = 1 / - Tan 45°- Cot 45° = 1 / -1- Cot 45° = -1

Jadi, Cot 315° bernilai -1.

25. c. Sin 210° +cos 300-cot (-315°)D. Sin (-240°) . Cos 330. sin 210°​


c. sin 210° + cos 300° - cot (-315°) = -1

d. sin (-240°) . Cos 330 . sin 210° = [tex]-\frac{3}{8}[/tex]

Simak pembahasan berikut.

Pembahasan

c. Sin 210° + cos 300° - cot (-315°)

Akan dicari satu persatu besar nilai perbandingan trigonometrinya.

sin 210°

sin 210° terletak pada kuadran III atau pada interval 180° ≤ x ≤ 270° dan memiliki nilai negatif. Maka besar sin 210° adalah

sin 210° = sin (180° + 30°)

sin 210° = -sin 30°

sin 210° = -[tex]\frac{1}{2}[/tex]

cos 300°

cos 300° terletak pada kuadran IV atau pada interval 270° ≤ x ≤ 360° dan memiliki nilai positif. Maka besar cos 300° adalah

cos 300° = cos (360° - 60°)

cos 300° = cos 60°

cos 300° = [tex]\frac{1}{2}[/tex]

cot (-315°)

Karena cot x = [tex]\frac{1}{tan x}[/tex], maka akan dicari nilai dari tan (-315°)

tan (-315°) = tan (360° - 315°)

tan (-315°) = tan 45°

tan (-315°) = 1

maka nilai cot (-315°) = 1

sin 210° + cos 300° - cot (-315°) = -[tex]\frac{1}{2}[/tex] + [tex]\frac{1}{2}[/tex] - 1

sin 210° + cos 300° - cot (-315°) = -1

d. sin (-240°) . Cos 330 . sin 210°

sin (-240°)

sin (-240°) = sin (360° - 240°)

sin (-240°) = sin 120°

sin 120° terletak pada kuadran II atau pada interval 90° ≤ x ≤ 180° dan memiliki nilai positif. Maka besar sin 120° adalah

sin 120° = sin (180° - 60°)

sin 120° = sin 60°

sin 120° = [tex]\frac{1}{2}[/tex]√3

cos 330°

cos 330° terletak pada kuadran IV atau pada interval 270° ≤ x ≤ 360° dan memiliki nilai positif. Maka besar cos 330° adalah

cos 330° = cos (360° - 30°)

cos 330° = cos 30°

cos 330° = [tex]\frac{1}{2}[/tex]√3

sin 210°

sin 210° terletak pada kuadran III atau pada interval 180° ≤ x ≤ 270° dan memiliki nilai negatif. Maka besar sin 210° adalah

sin 210° = sin (180° + 30°)

sin 210° = -sin 30°

sin 210° = -[tex]\frac{1}{2}[/tex]

sin (-240°) . Cos 330 . sin 210° = [tex]\frac{1}{2}[/tex]√3 × [tex]\frac{1}{2}[/tex]√3 × (-[tex]\frac{1}{2}[/tex])

sin (-240°) . Cos 330 . sin 210° = [tex]\frac{3}{4}[/tex] × (-[tex]\frac{1}{2}[/tex])

sin (-240°) . Cos 330 . sin 210° = [tex]-\frac{3}{8}[/tex]

34. Hitunglah nilai dari : a. Sin 315° + cot 315° b. cos 210° - sec 135° c. tan 240°× cosec 315°


a. sin 315 + cot 315 = (1/2)√2 + ( - 1 )

                                 =  √2 / 2  - 1

                                 = √2 / 2 - 2/2

                                 = (√2 - 2 ) / 2

b. cos 210 - sec 135 = - (1/2) √3 - ( - 2/√3 )

                                  = - √3 / 2 + 2 / √3

                                  = ( - 3 + 2√3 ) / 2√3

                                  = ( -3 + 2√3 ) / 2√3 x 2√3 / 2√3

                                  = ( -6√3 + 12 ) / 12

                                  = 6 ( - 2√3 + 2 ) / 12

                                  = (- 2√3 + 2 ) / 2

c. tan 240 x cosec 315 = √2 x ( - 2/√2 )

                                     = (- 2√2 ) / √2

                                     =( - 2√2 / √2 ) x √2 / √2

                                     = - 4 / 2

                                     = -2

40. Sin 300°.tan 310°.sec 420° / cot 135°.cos 210°.csc 315°


[tex] \frac{ \sin( {300}^{ \circ} ) \tan( {210}^{ \circ} ) \sec( {420}^{ \circ} ) }{ \cot( {135}^{ \circ} ) \cos( {210}^{ \circ} ) \csc( {315}^{ \circ} ) } = \boxed{ \bold{ \frac{1}{3} \sqrt{6} }} \\ [/tex]

Pembahasan

Nilai sin, cos, tan, cot, sec, csc pada sudut - sudut relasi trigonometri :

[tex] \sin(2\pi - x) = - \sin(x) \\ \tan(\pi + x) = \tan(x) \\ \cos(\pi + x) = - \cos(x) \\ \cos(2\pi + x) = \cos(x) \\ \\ \sec(x) = \frac{1}{ \cos(x) } \\ \csc(x) = \frac{1}{ \sin(x) } \\ \\ [/tex]

Maksud soal mungkin seperti ini :

[tex] \: \: \: \: \: \frac{ \sin( {300}^{ \circ} ) \tan( {210}^{ \circ} ) \sec( {420}^{ \circ} ) }{ \cot( {135}^{ \circ} ) \cos( {210}^{ \circ} ) \csc( {315}^{ \circ} ) } \\ \\ = \frac{ \sin{(360 - 60)}^{ \circ} . \tan{(180 + 30)}^{ \circ}. \sec{(360 + 60)}^{ \circ} }{ \cot {(180 - 45)}^{ \circ}. \cos {(180 + 30)}^{ \circ} . \csc {(360 - 45)}^{ \circ} } \\ \\ = \frac{( - \frac{1}{2} \sqrt{3} )( \frac{1}{3} \sqrt{3} )(2) }{( - 1)( - \frac{1}{2} \sqrt{3})( - \sqrt{2} ) } \\ \\ = \frac{1}{ \frac{1}{2} \sqrt{6} } \\ \\ = \frac{2}{ \sqrt{6} } \\ \\ = \frac{1}{3} \sqrt{6} \\ \\ [/tex]
