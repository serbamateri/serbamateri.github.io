9. 3. Buatlah algoritma untuk menghitung gaji bersih seorang pegawai PT. Adi Pratama pada bulan Februari 2020. Gaji bersih karyawan terdiri atas gaji pokok, tunjangan, transportasi, dan ada pajak juga yang harus dibayarkan. Gaji pokok diinputkan oleh user yang akan menghasilkan output tunjangan, transportasi, pajak dan gaji bersihnnya. Rumusnya sebagai berikut. Tunjangan = 20% dari gaji pokok Transportasi = 5% dari gaji pokok Pajak = 2% dari gaji pokok Gaji Bersih Karyawan - pajak transportasi pesert gaji pokok + tunjangan +​


Penjelasan:

Berikut adalah algoritma program untuk menghitung gaji bersih seorang pegawai PT. Adi Pratama pada bulan Februari 2020:

Input gaji pokok

Hitung tunjangan = gaji pokok x 20%

Hitung transportasi = gaji pokok x 5%

Hitung pajak = gaji pokok x 2%

Hitung gaji bersih = gaji pokok + tunjangan + transportasi - pajak

Output tunjangan, transportasi, pajak, dan gaji bersih

Di bawah ini adalah contoh program implementasi algoritma di Python:

# Input gaji pokok

gaji_pokok = int(input("Masukkan gaji pokok: "))

# Hitung tunjangan

tunjangan = gaji_pokok * 0.2

# Hitung transportasi

transportasi = gaji_pokok * 0.05

# Hitung pajak

pajak = gaji_pokok * 0.02

# Hitung gaji bersih

gaji_bersih = gaji_pokok + tunjangan + transportasi - pajak

# Output tunjangan, transportasi, pajak, dan gaji bersih

print("Tunjangan: ", tunjangan)

print("Transportasi: ", transportasi)

print("Pajak: ", pajak)

print("Gaji Bersih: ", gaji_bersih)

Berikut adalah contoh program sederhana dalam bahasa pemrograman Python yang menghitung tunjangan jabatan, tunjangan keluarga, dan gaji bersih berdasarkan informasi yang diberikan:

```python

# Informasi pegawai

pegawai1 = {

'nama': 'Ahmad Muslim',

'agama': 'Islam',

'gaji_pokok': 4000000,

'jumlah_anak': 2

}

pegawai2 = {

'nama': 'Alex',

'agama': 'Kristen Protestan',

'gaji_pokok': 6000000,

'jumlah_anak': 5

}

# Fungsi untuk menghitung tunjangan jabatan

def hitung_tunjangan_jabatan(gaji_pokok):

return 0.2 * gaji_pokok

# Fungsi untuk menghitung tunjangan keluarga

def hitung_tunjangan_keluarga(gaji_pokok, jumlah_anak):

if jumlah_anak <= 2:

return 0.1 * gaji_pokok

else:

return 0.2 * gaji_pokok

# Fungsi untuk menghitung gaji bersih

def hitung_gaji_bersih(gaji_pokok, tunjangan_jabatan, tunjangan_keluarga):

return gaji_pokok + tunjangan_jabatan + tunjangan_keluarga

# Menampilkan informasi pegawai dan hasil perhitungan

def cetak_info_pegawai(pegawai):

print(f'Nama Pegawai: {pegawai["nama"]}')

print(f'Agama: {pegawai["agama"]}')

print(f'Gaji Pokok: Rp. {pegawai["gaji_pokok"]:,}')

tunjangan_jabatan = hitung_tunjangan_jabatan(pegawai["gaji_pokok"])

print(f'Tunjangan Jabatan: Rp. {tunjangan_jabatan:,}')

tunjangan_keluarga = hitung_tunjangan_keluarga(pegawai["gaji_pokok"], pegawai["jumlah_anak"])

print(f'Tunjangan Keluarga: Rp. {tunjangan_keluarga:,}')

gaji_bersih = hitung_gaji_bersih(pegawai["gaji_pokok"], tunjangan_jabatan, tunjangan_keluarga)

print(f'Gaji Bersih (Take Home Pay): Rp. {gaji_bersih:,}')

# Menampilkan informasi pegawai

print("Informasi Pegawai 1:")

cetak_info_pegawai(pegawai1)

print("\nInformasi Pegawai 2:")

cetak_info_pegawai(pegawai2)

```

Program ini akan menghitung tunjangan jabatan, tunjangan keluarga, dan gaji bersih untuk kedua pegawai berdasarkan informasi yang diberikan dan kemudian mencetak hasilnya. Pastikan Anda menjalankan program ini di lingkungan Python yang sudah terinstal di komputer Anda.

A. Akuntan Publik

  Akuntan publik berperan dalam penyajian laporan informasi keuangan perusahaan baik yang akan berencana untuk go public maupun perusahaan yang telah mencatatkan sahamnya (listed) di bursa.   Profesi lain pihak jasa profesi lain yang dapat memberikan pendapat atau penilaian sesuai dengan perkembangan pasar modal di masa mendatang dan terdaftar di Otoritas Jasa Keuangan.

Buatlah program untuk menghitung gaji pegawai Input : nama, golongan Proses : · Jika golongan A maka gaji pokok =Rp 1.500.000, tunjangan =5% * gaji pokok .jika gol.B maka gaji pokok = Rp.1.700.000,tunjangan = 10%* gaji pokok .jika gol.C maka gaji pokok = Rp.2.000.000, tunjangan = 15%*gaji pokok Output : Nama,gaji pokok,tunjangan serta gaji bersih Petunjuk : gunakan array dan petualangan


program untuk menghitung gaji pegawai, program ditulis dalam bahasa C

#include <stdio.h>

int main()

{

    char nama[50];

    char golongan;

    int gaji[3] = { 1500000, 1700000, 2000000 };

    int persentunjangan[3] = { 5, 10, 15 };

    int gajibersih, tunjangan;

    printf("Masukan nama Anda : ");

    scanf("%s", nama);

    printf("Masukan golongan (A, B, atau C) : ");

    scanf("%c", &golongan);

    golongan = golongan - 'A';

    tunjangan = gaji[ golongan ] * persentunjangan[ golongan ] / 100;

    gajibersih = gaji[ golongan ] + tunjangan;

    printf("Nama : %s\n", nama);

    printf("Gaji pokok : %i\n", gaji[ golongan ]);

    printf("Tunjangan : %i\n", tunjangan);

    printf("Gaji bersih : %i\n", gajibersih);

    return 0;

}

Pembahasan

Ada bagian yang aneh pada soal, "gunakan array dan petualangan". Array adalah bentuk variabel dengan alokasi memory untuk banyak nilai. Yang saya bingung adalah istilah "petualangan" disini. Setahu saya tidak ada istilah "petualangan" dalam bahasa pemrograman, mohon dikoreksi jika memang terjadi typo.

Pada solusi pemrograman yang dibuat, kita memanfaatkan golongan untuk menjadi index dari Array.

Pada potongan code berikut golongan = golongan - 'A';

Berfungsi untuk mengubah karakter A menjadi nilai 0. Perlu diketahui karakter adalah sebuah huruf yang memiliki nilai angka pada komputer, atau ASCII. Sehingga bisa dioperasikan menggunkana operator aritmatika seperti penjumlahan pengurangan, dan lainnya.

Maka jika golongan adalah A dan dikurangi 'A' maka hasilnya adalah 0

jika golongan adalah B dan dikurangi 'A' maka hasilnya adalah 1

jika golongan adalah C dan dikurangi 'A' maka hasilnya adalah 2

Nilai ini bisa kita gunakan sebagai index dalam array gaji yang sudah didefinisikan sebelumnya.

Sehingga kita bisa akses secara dinamis array gaji menggunakan golongan sebagi indexnya.

Jawab:

Arinda akan menerima gaji dan tunjangan sebesar Rp2.675.000,00 setiap bulan.

Penjelasan dengan langkah-langkah:

Untuk menghitung besar gaji dan tunjangan yang diterima Arinda setiap bulan, berikut adalah langkah-langkahnya:

Hitung tunjangan keluarga:

Tunjangan keluarga = 1/5 x gaji pokok + Rp50.000,00

Tunjangan keluarga = 1/5 x Rp2.000.000,00 + Rp50.000,00

Tunjangan keluarga = Rp400.000,00 + Rp50.000,00

Tunjangan keluarga = Rp450.000,00

Hitung tunjangan kesehatan:

Tunjangan kesehatan = 1/2 x tunjangan keluarga

Tunjangan kesehatan = 1/2 x Rp450.000,00

Tunjangan kesehatan = Rp225.000,00

Hitung total gaji dan tunjangan:

Total gaji dan tunjangan = gaji pokok + tunjangan keluarga + tunjangan kesehatan

Total gaji dan tunjangan = Rp2.000.000,00 + Rp450.000,00 + Rp225.000,00

Total gaji dan tunjangan = Rp2.675.000,00

Jadi, Arinda akan menerima gaji dan tunjangan sebesar Rp2.675.000,00 setiap bulan.
