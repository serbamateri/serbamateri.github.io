Jawaban:Kunci dihapus secara permanen sekitar 10 menit setelah menerima pesan dan tidak disimpan di server. Dengan transisi ke protokol MTProto 2.0, algoritma hash kriptografi SHA-256 digunakan.

Mode obrolan rahasia menggunakan jenis enkripsi end-to-end yang lebih dapat diandalkan, di mana kunci untuk membaca pesan sama sekali tidak dikirim ke server dan hanya tersedia untuk pengirim dan penerima. Untuk mengimplementasikan fungsi ini, algoritma AES-256 dalam mode IGE digunakan.

Perhatian! Ketika Anda menghapus pesan dalam korespondensi, itu juga dihapus secara permanen dari server.

Kode Sumber Telegram

Sumber terbuka Telegram adalah kode API klien terbuka yang tersedia untuk diunduh gratis. Terlepas dari kenyataan bahwa bagian server dari kode tetap tertutup, semua fungsi messenger tetap berfungsi.

Tujuan

Berkat antarmuka pemrograman aplikasi terbuka, setiap orang yang berminat dapat mengakses perpustakaan data TDLib dan membuat klien web sendiri atas dasar itu. Pada saat yang sama, Anda tidak perlu beradaptasi dengan bahasa pemrograman apa Telegram ditulis, karena beberapa versi dengan dukungan Javascript, C ++ dan C # ditawarkan sekaligus.

Perpesanan, enkripsi, dan basis data lokal akan berfungsi secara identik dengan Telegram asli. Kemampuan untuk mengedit kode diberikan untuk mempopulerkan kebijakan perusahaan dan mengembangkan kebebasan berbicara, perlindungan hak asasi manusia dan anonimitas komunikasi. Ini juga memberikan insentif untuk pengembangan teknologi baru di bidang komunikasi dan media massa.

Cara Penggunaan

Sebelumnya, untuk membuat klien Telegram, kode sumber harus diunduh dari situs resmi telegram.org atau di sumber lain. Anda juga perlu membaca instruksi terlampir pada enkripsi data. Kemudian Anda dapat mulai bekerja (secara alami, tanpa pengetahuan khusus dan keterampilan pemrograman, tidak ada yang akan dilakukan).

Referensi! Tidak mungkin mengunduh kode untuk pembaruan aplikasi terbaru, karena tidak pernah dipublikasikan. Hanya versi lama dari klien resmi untuk semua platform yang tersedia untuk diunduh.

Klien web khusus dapat ditulis menggunakan bahasa dan kerangka kerja pemrograman apa pun, sehingga pilihan Anda tidak terbatas.
