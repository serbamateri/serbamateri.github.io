Bruto = Nilai/berat kotor pada barang

Tarra = Nilai/berat kemasan

RUMUS

Tarra = %Tarra x Bruto

Netto = Bruto - Tarra

Tarra = 2% x 42

Tarra = [tex]\frac{2}{100} \times 42[/tex]

Tarra = 0,84 kg

Netto = 42 Kg - 0,84 kg

28. Berat 1 kaleng cat adalah x kg, setelah dipakai 3/5 kg berat menjadi y, maka berat kosong kaleng cat tersebut adalah


berat kaleng = x kg
dipakai 3/5 kg berat menjadi y,maka
x - 3/5 = y
x = 3/5 + y

berat kosong = x - (3/5+y)

Diketahui :

Berat 1 kaleng cat = x kg

Berat cat setelah dipakai 3/5 kg berat = y kg


Ditanya: berat kosong kaleng cat tersebut ....


Penyelesaian :

Berat 1 kaleng cat = kaleng + cat mula-mula = x

Berat cat setelah dipakai 3/5 kg berat : y = Berat cat mula-mula - [tex] \frac{3}{5} [/tex]

Jadi, berat cat mula-mula = y + [tex] \frac{3}{5} [/tex]


Jika x = cat mula-mula + kaleng, maka :

x = cat mula-mula + kaleng

x = y + [tex] \frac{3}{5} [/tex] + kaleng

kaleng = x - ( y + [tex] \frac{3}{5} [/tex] )

kaleng = x - y - [tex] \frac{3}{5} [/tex] kg


Jadi, berat kosong kaleng tersebut = x - y - [tex] \frac{3}{5} [/tex] kg
