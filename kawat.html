9. kawat p dan kawat Q memiliki diameter yang sama. namun panjang kawat p hanya 1/4 panjang kawat Q. tentukan hasil bagi hambatan kawat p dengan hamabtan kawat Q?


Perbandingan hambatan P dan hambatan Q adalah [tex]\boxed{ 1\: : \: 4}[/tex]


Pembahasan.

Hambatan Jenis merupakan kecenderungan suatu bahan untuk menahan aliran arus listrik.

Hambatan jenis sebanding dengan Besarnya hambatan dan luas permukaan kawat, tetapi berbanding terbalik dengan panjang kawat.

Hambatan jenis dapat ditentukan dengan persamaan :

[tex]\boxed{ \rho = \frac{R \: A}{L}  }[/tex]

Ket :

ρ = hambatan jenis (Ω m)

R = Hambatan (Ω)

A = Luas Pemukaan (m²)

L = Panjang kawat (m)


Diketahui :

Diameter p (dp) = diameter Q (dq)

Sehingga Ap = Aq

Panjang P (Lp) = 1/4 Panjang Q (Lq)


Ditanya:

Rp : Rq


Penyelesaian:

Dengan menggunakan persamaan :

[tex]\boxed{ \rho = \frac{R \: A}{L}  }[/tex]

dimana

ρp = ρq

[tex]\frac{R_{p} \: A_{p}  }{L_{p} }  = \frac{R_{q} \: A_{q}  }{L_{q} }[/tex]

[tex]\frac{R_{p} }{R_{q} }  = \frac{L_{p} }{L_{q} }[/tex]

Rp : Rq = 1/4 : 1

Rp : Rq = 1 : 4


Kesimpulan.

Perbandingan hambatan P dan hambatan Q adalah 1 : 4

12. Dua kawat terbuat dari bahan yang sama. Panjang kawat kedua 4kali panjang kawat pertama dan diameter kawat kedua 2kali diameter kawat pertama. Jika hambatan kawat pertama adalah R, maka hambatan kawat kedua adalah


Hambatan kawat penghantar

Perbandingan luas penampangnya:
A : A2 = π/4d² : π/4(2d)² = 1 : 4
A2 = 4A

Hambatan pertama:
R = α L / A

Hambatan kedua:
R2 = α L2 / A2
R2 = α 4L / 4A

Perbandingannya:
R/R2 = (α L / A) / (α 4L / 4A)
R/R2 = 1 / 1

R = R2

Jadi, hambatan kawat kedua sama dengan hambatan kawat pertama.

tegangan
σ = F/A = 40/(8 x 10^-6)
σ = 5 x 10^6 N/m²

regangan
ε = Δι/ι = (0.4 x 10^-3)/1.2 = 3.3 x 10^-4

modulus elastisitas
γ = σ/ε
γ = (5 x 10^6)/(3.3 x 10^-6) = 1.5 x 10^12 N/m²

[tex]\displaystyle \frac{R_A}{R_B}=\frac{\frac{4\rho L}{\pi D_A^2}}{\frac{4\rho L}{\pi D_B^2}}[/tex]
[tex]\displaystyle \frac{R_A}{R_B}=\left ( \frac{D_B}{D_A} \right )^2[/tex]
[tex]\displaystyle \frac{R_A}{R_B}=\left ( \frac{D_B}{1/2~D_B} \right )^2[/tex]
[tex]\displaystyle R_A:R_B=1:\frac{1}{4}\\ R_A:R_B=4:1[/tex]

Diket:
D1 = 2.D2
R = ρl/A

karena ρ dan l sama, maka:

R1/R2 = A2 / A1
R/R2 = (D2)² / (D1)²
R/R2 = (D2)² / (2.D2)²
R/R2 = D2² /4.D2²
R/R2 = 1/4
R2 = 4R

jadi hambatan ke2 = 4 kali hambatan pertama.
