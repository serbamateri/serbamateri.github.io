6. Berapa massa perak yang akan diendapkan jika 5,0 gram tembaga di reaksikan dengan larutan perak nitrat berlebih? Cu(s)+2Agno3(aq)—>Cu(No3)2(aq)+2Ag(s)


Berapa massa perak yang akan diendapkan jika 5,0 gram tembaga di reaksikan dengan larutan perak nitrat berlebih?

Cu (s) + 2AgNo₃ (aq) → Cu(No₃)₂ (aq) + 2Ag (s)

⇒ Massa perak yang akan diendapkan adalah 17,28 gram.

Penjelasan dengan langkah-langkah:

Diketahui:

Massa tembaga 5,0 gram.

Persamaan reaksinya:

Cu (s) + 2AgNo₃ (aq) → Cu(No₃)₂ (aq) + 2Ag (s)

Ditanya:

Berapa massa perak yang akan diendapkan?

Jawab:

Persamaan reaksinya:

Cu (s) + 2AgNo₃ (aq) → Cu(No₃)₂ (aq) + 2Ag (s)

Mencari mol Cu.

Ar Cu = 63,5

mol Cu = massa ÷ Ar Cu

mol Cu = 5 ÷ 63,5

mol Cu = 0,08 mol

Mencari mol Ag.

mol Ag = koefisien Ag ÷ koefisien Cu x mol Cu

mol Ag = 2 ÷ 1 x 0,08

mol Ag = 2 x 0,08

mol Ag = 0,16 mol

Mencari massa Ag.

massa Ag = mol Ag x Ar Ag

massa Ag = 0,16 x 108

massa Ag = 17,28 gram

Jadi, massa perak yang akan diendapkan jika 5,0 gram tembaga di reaksikan dengan larutan perak nitrat berlebih adalah 17,28 gram.

Pembahasan :

Cu (s) + HNO₃ (aq) → Cu(NO₃)₂ (aq) + NO (aq) + H₂O (l)

  |_______________|

 0                               +2     (Reaksi Oksidasi/ Reduktor)

Atom Cu pada reaksi diatas mengalami kenaikan bilangan oksidasi dari 0 berubah menjadi +2, sehingga mengalami reaksi oksidasi atau reduktor.

Cu (s) + HNO₃ (aq) → Cu(NO₃)₂ (aq) + NO (aq) + H₂O (l)

                |____________________|

               +5                                         +2    (Reaksi Reduksi/ Oksidator)

Atom N pada reaksi diatas mengalami penurunan bilangan oksidasi dari +5 berubah menjadi +2, sehingga mengalami reaksi reduksi atau oksidator.

Maka diperoleh :

Oksidator = mengalami reduksi = biloksnya turun = HNO₃

Reduktor = mengalami oksidasi = biloksnya naik = Cu

Hasil Oksidasi = Cu(NO₃)₂

Hasil Reduksi = NO

18. Setarakan persamaan reaksi berikut : Cu(s) + HNO3(aq) → Cu(NO3)2(aq) + H2O(l) + NO(g)


Reaksi kimia merupakan salah satu proses di mana satu zat atau lebih diubah menjadi zat yang berbeda dan akan menghasilkan suatu produk yang baru. Berdasarkan soal, dapat disimpulkan bahwa penyetaraan reaksi Cu (s) + [tex]HNO_{3}[/tex] (aq) → [tex]Cu(NO_{3}) _{2}[/tex] (aq) + [tex]H_{2}O[/tex] (l) + NO (g) adalah 3Cu (s) + [tex]8HNO_{3}[/tex] (aq) → [tex]3Cu(NO_{3}) _{2}[/tex] (aq) + [tex]4H_{2}O[/tex] (l) + 2NO (g).

Pembahasan:

Pada saat kita mempelajari mata pelajaran Kimia maka kita akan belajar tentang reaksi kimia. Reaksi kimia merupakan salah satu proses di mana satu zat atau lebih diubah menjadi zat yang berbeda dan akan menghasilkan suatu produk yang baru. Untuk menyetarakan reaksi kimia, bisa menggunakan metode bilangan oksidasi (biloks) dan metode setengah reaksi. Penyetaraan reaksi dengan metode biloks dari Cu (s) + [tex]HNO_{3}[/tex] (aq) → [tex]Cu(NO_{3}) _{2}[/tex] (aq) + [tex]H_{2}O[/tex] (l) + NO (g) yaitu:

Cu (s) + [tex]HNO_{3}[/tex] (aq) → [tex]Cu(NO_{3}) _{2}[/tex] (aq) + [tex]H_{2}O[/tex] (l) + NO (g)

↓           ↓ ↓  ↓             ↓        ↓               ↓  ↓        ↓  ↓

0         +1 +5 -6            +2      -2            +2 -2      +2 -2

Oksidasi: Cu →  [tex]Cu(NO_{3}) _{2}[/tex], dengan biloks 0 menjadi 2 sehingga terjadi 2.

Reduksi: [tex]HNO_{3}[/tex] → NO, dengan biloks +5 menjadi +2 sehingga terjadi penurunan 3. Maka senyawa yang mengalami oksidasi x3 sedangkan senyawa yang mengalami reduksi x2, maka:

3Cu (s) + [tex]4HNO_{3}[/tex] (aq) → [tex]3Cu(NO_{3}) _{2}[/tex] (aq) + [tex]4H_{2}O[/tex] (l) + 2NO (g).

Samakan jumlah unsur pada ruas kiri dan kanan, jumlah hidrogen ruas kiri 8 sedangkan ruas kanan 4 maka hidrogen pada ruas kanan x2 sehingga reaksinya menjadi:

3Cu (s) + [tex]8HNO_{3}[/tex] (aq) → [tex]3Cu(NO_{3}) _{2}[/tex] (aq) + [tex]4H_{2}O[/tex] (l) + 2NO (g).

Dengan demikian, penyetaraan reaksi Cu (s) + [tex]HNO_{3}[/tex] (aq) → [tex]Cu(NO_{3}) _{2}[/tex] (aq) + [tex]H_{2}O[/tex] (l) + NO (g) adalah 3Cu (s) + [tex]8HNO_{3}[/tex] (aq) → [tex]3Cu(NO_{3}) _{2}[/tex] (aq) + [tex]4H_{2}O[/tex] (l) + 2NO (g).

Aturan-aturan untuk menentukan Bilangan Oksidasi

• Unsur bebas mempunyai biloks =0 Contoh → H,N,Fe berturut-turut dalam H2, N2, dan Cl= 0

• Fluorin, unsur yang mempunyai biloks -1

• Golongan I (logam alkali ) biloks = +1 contoh: Li, Na, K, Rb, Cs

• Golongan II (Alkali Tanah) biloks = +2 contoh = Be, Mg, Ca, Sr, Ba

• biloks suatu unsur dalam suatu ion tunggal sama dengan muatannya Contoh: biloks Fe dalam ion Fe²+ = +2 S²- = 2-

• Biloks H umumnya= +1, kecuali dalam senyawa dengan logam maka biloks = -1 Contoh: NaH BaH2 = -1

• biloks O umunya = -2 Contoh: biloks O dalam H2O, MgO = -2 kecuali
-Dalam Fe2O = +2
-Dalam Peroksida, seperti H2O2, biloks O = -1 -Dalam Senyawa superoksida, seperti KO2 biloks O= -1/2
-Dalam senyawa terner, biloks O selalu = -2
-Dalam senyawa biner, biloks O belum tentu = +2

• biloks halogen, sperti F, Cl, I, Br, At = -1

• biloks O dalam unsur-unsur dalam suatu senyawa = 0 Contoh: H2SO4= 0

• bilols unsur-unsur dalam senyawa poliatom = muatannya contoh = SO4²- = -2

• biloks dalam senyawa ion = muatan kation dan anionya.


39. Perhatikan reaksi berikut. Cu(s) + 4HNO3(aq) -> Cu(NO3)2(aq) + 2NO2(g) + 2H2O(l) Zat yang bertindak sebagai oksidator adalah . . . . a. Cu b. HNO3 c. Cu(NO3)2 d. NO2 e. H2O


Pada reaksi berikut : Cu (s) + 4 HNO₃ (aq) → Cu(NO₃)₂ (aq) + 2 NO₂ (g) + 2  H₂O (l) zat yang bertindak sebagai oksidator adalah HNO₃ (B).

Pembahasan :

(Reaksi Oksidasi/ Reduktor)

Cu (s) + 4 HNO₃ (aq) → Cu(NO₃)₂ (aq) + 2 NO₂ (g) + 2  H₂O (l)

|_________________|

0                                   +2

Atom Cu pada reaksi diatas mengalami kenaikan bilangan oksidasi dari 0 berubah menjadi +2, sehingga mengalami reaksi oksidasi atau reduktor.

(Reaksi Reduksi/ Oksidator)

Cu (s) + 4 HNO₃ (aq) → Cu(NO₃)₂ (aq) + 2 NO₂ (g) + 2  H₂O (l)

                   |______________________|

                  +5                                             +4

Atom N pada reaksi diatas mengalami penurunan bilangan oksidasi dari +5 berubah menjadi +4, sehingga mengalami reaksi reduksi atau oksidator.

Maka diperoleh :

Oksidator = mengalami reduksi = biloksnya turun = HNO₃

Reduktor = mengalami oksidasi = biloksnya naik = Cu

Hasil Oksidasi = Cu(NO₃)₂

Hasil Reduksi = NO₂

Oksidator dan Reduktor

Oksidator adalah zat yang mengoksidasi zat lain, sedangkan zat itu sendiri mengalami reduksi.

Reduktor adalah zat yang mereduksi zat lain, sedangkan zat itu sendiri mengalami oksidasi.
