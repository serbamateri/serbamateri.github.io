⟩⟩ Dekat atau mendekati

Contoh kalimat “NEAR”

I am near the gas station.They wishes they lived near the restaurants and banks.She is looking for a hotel near the airport.We want to go to Japan near the end of this month.Hani is going to buy individual health insurance in the near future.

15. Please help me, liat tutor di yt makin ga paham :')​


Bilangan kompleks dapat dinyatakan ke dalam bentuk polar |z|(cos(t) + i.sin(t)), di mana |z| modulus dari bilangan kompleks z.

4.a)

Misalkan z = 2 + 2√(3)i

|z| = √(2^2 + (2√3)^2)

= √(4 + 12)

= √(16)

= 4

arctan((2√(3))/2) = arctan(√3) = pi/3 (kuadran 1, karena bagian real dan imajinernya positif)

Jadi, bentuk polarnya adalah 4(cos(pi/3) + i.sin(pi/3)).

===

4.b)

Misalkan z = -5 + 5i

|z| = √((-5)^2 + 5^2)

= √(25 + 25)

= √(25.2)

= 5√2

arctan(5/(-5)) = arctan(-1) = (3pi)/4 (kuadran 2, karena realnya negatif, tetapi imajinernya positif)

Jadi, bentuk polarnya adalah (5√2)(cos((3pi)/4) + i.sin((3pi)/4))

===

5.a)

|z| = √((-1)^2 + (1)^2) = √2

arctan(1/(-1)) = arctan(-1) = (3pi)/4 (kuadran 2, karena realnya positif, imajinernya negatif)

maka, bentuk tersebut dapat diubah menjadi:

((√2)(cos((3pi)/4) + i.sin((3pi)/4)))^(1/3)

Untuk mencari akar-akarnya, pangkatkan |z| dengan (1/n), sehingga (√2)^(1/3) = 2^(1/6).

Selanjutnya, misalkan t = (3pi/4 + 2.pi.k)*(1/3)

= (pi/4 + (2/3).pi.k)

= (3pi + 8.pi.k)/12, dengan k = 0, 1, 2.

Jika k = 0, maka (3pi + 8.pi.k)/12 = pi/4

Jika k = 1, maka (3pi + 8.pi.k)/12 = (11.pi)/12

Jika k = 2, maka (3pi + 8.pi.k)/12 = (19.pi)/12 = ((19 - 24)pi)/12 = ((-5).pi)/12

Jadi, akar-akarnya adalah

x_1 = (2^(1/6))(cos(pi/4) + i.sin(pi/4))

x_2 = (2^(1/6))(cos(11.pi/12) + i.sin(11.pi/12))

x_3 = (2^(1/6))(cos(-5.pi/12) + i.sin(-5.pi/12))

Untuk lokasinya, tinggal hitung nilai derajat dalam sin dan cos dan cari tahu apakah nilainya akan bernilai negatif atau positif. Hasilnya adalah:

x_1 berada di kuadran 1, x_2 berada di kuadran 2, dan x_3 berada di kuadran 4.

===

5.b)

|z| = √((-2√3)^2 + (-2)^2) = √(12 + 4) = √(16) = 4

arctan((-2)/((-2√3)) = arctan(1/√(3)) = -5pi/6 (kuadran 4, karena real dan imajinernya negatif)

maka, bentuk tersebut dapat diubah menjadi:

((4)(cos((-5pi)/6) + i.sin((-5pi)/6))))^(1/4)

Untuk mencari akar-akarnya, pangkatkan |z| dengan (1/n), sehingga (4)^(1/4) = √2

Selanjutnya, misalkan t = (-5pi/6 + 2.pi.k)*(1/4)

= (-5pi/24 + (pi.k)/2)

= (-5pi + 12pi.k)/24

Jika k = 0, maka (-5pi + 12pi.k)/24 = (-5pi)/24

Jika k = 1, maka (-5pi + 12pi.k)/24 = (7pi)/24

Jika k = 2, maka (-5pi + 12pi.k)/24 = (19pi)/24

Jika k = 3, maka (-5pi + 12pi.k)/24 = (31pi)/24 = (31 - 48)pi/24 = (-17pi)/24

Jadi, akar-akarnya adalah:

x_1 = √2(cos((-5pi)/24) + i.sin((-5pi)/24)) di kuadran 4

x_2 = √2(cos((7pi)/24) + i.sin((7pi)/24) di kuadran 1

x_3 = √2(cos((19pi)/24) + i.sin((19pi)/24) di kuadran 2

x_4 = √2(cos((-17pi)/24) + i.sin((-17pi)/24) di kuadran 3

21. tutor kak no 31tutor​


8 cm

Penjelasan dengan langkah-langkah:

Diketahui panjang AB, BC, CD, dan DE adalah sama yaitu 4 cm

.

Cari nilai AC :

AC² = AB² + BC²

AC² = 4² + 4²

AC² = 16 + 16

AC² = 32

AC = √32

AC = 4√2 cm

.

Cari nilai AD :

AD² = AC² + CD²

AD² = (4√2)² + 4²

AD² = 32 + 16

AD = √48

AD = √16×3

AD = 4√3 cm

.

Panjang AE :

AE² = AD² + DE²

AE² = (4√3)² + 4²

AE² = 48 + 16

AE = √64

AE = 8 cm
