14. langkah-langkah menginstal modem dial-up


Masukkan CD driver modem, tunggulah beberapa saat hingga muncul sebuah ventilasi instalasi. Biasanya CD driver seperti ini memiliki layanan autorun sehingga begitu anda memasukkan CD driver maka ventilasi instalasi pun akan segera muncul.Selanjutnya pilihlah jenis modem nan terpasang di komputer anda, pilih PCI modem buat modem internal dan pilih USB modem jika modem nan anda pakai terhubung ke port USB.Tahap berikutnya ialah memilih perintah instalmodem driver .Setelah itu tentukan sistem operasi nan berjalan di komputer anda.Kemudian tunggulah hingga proses instalasi berjalan hingga menunjukan indikasi selesai.Sekarang periksa apakah modem tersebut telah ter- instal dengan benar. Caranya, pilih tombolstart dan pilih opsi all programs . Kemudian, lanjutkan dengan melakukan klik tombol kananmouse pada opsi my computer lalu pilihlah opsiproperties . Di sebuah ventilasi nan muncul, masuklah ke dalam tab hardware dan pilihdevice manager . Di bagian inilah, jika modem telah ter- instal dengan baik maka akan terlihat jenis modem nan ada di komputer anda.Berikutnya, lakukan setting berkaitan dengan koneksi dial up nan hendak anda pakai. Pada contoh ini koneksi dial up yang akan dipakai ialah koneksi telkomnet instan. Untuk itu pilihlah tombol start di komputer anda, lalu pilih opsi all programs lanjutkan dengan memilh opsiaccessories lalu opsi communications dan opsinew connections wizard .Pada ventilasi nan muncul, pilihlah opsi next .Lalu pilih opsi connect to the internet , lalu klik tombol next .Lantas pilihlah opsi set up my connections manually , lalu klik tombol next .Lanjutkan dengan memilih opsi connect using dial up modem , kemudian klik tombol next .Masukkan nama internet service provider di kolom isp name . Misalnya, telkomnet. Lantas klik tombol next .Ketikkan nomor telepon di kolom phone numberlalu klik tombol next .Masukkan informasi nan berkaitan dengan layanan internet nan meliputi; username : telkomnet@instan dan password : telkom. Ulangi penulisan password di kolom confirm password lalu klik tombol next .Pada tampilan ventilasi berikutnya, klik tombolfinish buat mengakhiri proses setting ini.Sekarang, anda sudah dapat melakukan dial ke layanan telkomnet instan. Caranya, pilihlah tombol start di komputer anda lalu pilih opsiconnection dan lanjutkan dengan memilih opsi telkomnet@instan. Pada ventilasi connect telkom nan muncul, isikan username : telkomnet@instan dan password : telkom lalu klik tombol dial .

26. Apakah fungsi utama penginstalan modem


Apakah fungsi utama penginstalan modemJawaban Pendahuluan

Sebelum membahas mengenai fungsi utama penginstallan modem, mari kita bahas terlebih dahulu apa itu modem. Modem merupakan komponen dari komputer yang berfungsi agar komputer kita bisa terhubung atau mengakses internet.

Pembahasan

Setelah mengetahui apa itu modem, maka kita bisa menjelaskan mengenai fungsi dari penginstallan modem. Agar bisa terhubung atau mengakses internet, maka modem perlu diinstall terlebih dahulu agar terbaca oleh komputer dan bisa diinputkan beberapa parameter agar komputer kita terhubung dengan internet. Sering dengan perkembangan jaman, modem yang semula berupa modem internal yang terpasang pada motherboard sudah berubah menjadi modem USB.

Kesimpulan

Demikian pembahasan mengenai fungsi utama modem, yaitu agar modem terdeteksi oleh komputer dan bisa kita gunakan untuk mengakses internet. Namun modem pada jaman sekarang juga sudah mulai tergantikan dengan smartphone. Hal ini dikarenakan smartphone juga sudah bisa dijadikan sebagai modem melalui menu hotspot portbale dan tethering. Untuk mempelajari materi lainnya mengenai modem, perangkat internet dan materi lainnya dapa dibaca pada link berikut

Langkah pertama Internet setting adalah melakukan setting di komputer untuk akses ke layanan SPEEDY.
Untuk setting di komputer dilakukan sesuai dengan operating system yang dipakai di komputer pelanggan, disini yang dibahas khusus untuk Windows XP.Lakukan network setting di PC (computer) sbb :
Start – Control Panel – Network Connection, sehingga keluar menu berikut ini :

36. proses penginstalan pada modem biasanya menggunakan adalah


Modem Internal :
Modem internal sesuai dengan namanya dipasang di dalam komputer. Modem ini berbentuk kartu (PCI atau ISA) yang dipasang pada slot ekspansi pada mainboard komputer.

Modem eksternal :
Modem eksternal sesuai namanya memang terpasang eksternal (di luar) komputer dan terhubung pada salah satu port serial (COM 1 atau COM 2) pada panel belakang CPU.Modem eksternal mempunyai catudaya tersendiri sehingga tidak mengambil dari power supply komputer.

Cara Menginstal Modem di Laptop
1. Buka Windows Explorer dan perhatikan isi modem, perangkat keras komputer modem bisa pula berfungsi sbg media penyimpanan.
2. Selanjutnya klik kanan dan pilih Install or run program from your media, sehingga kotak dialog Setup dapat ditampilkan.
3. Yang harus dilakukan sekarang adalah menekan tombol Next , maka kotak dialog setup akan di tampilkan lagi.
4. Dalam tahap ini, anda akan diminta untuk menyetujui penginstallan , jika anda setutu klik I Agree, dikotak dialog dapat diberitahukan penyimpampanan file-file instalasi modem, anda dapat menetukan tempat penyimpanan file instalasi sendiri.
5. Klik tombol instal , tunggula beberapa saat... dan tunggu hingga proses intalasi selesai sehingga kotak dialaog setup akan ditampilkan dan  memberitahukan bahwa proses intsalasi modem sudah berhasil.
6. Klik Finish dan modem telah terinstal di perangkat anda. Sekarang anda bisa menggunakan modem tersebut untuk berinternet.
