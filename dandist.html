Luas trapesium?

Luas = 1/2×(a+b)×t = 1/2×(4+7,2)×6 = 11,2×3 = 33,6m²

Panjang lantai rumah = 7m

Lebar lantai rumah = 6m

Ditanya :

Harga seluruh ubin?

Jawab :

Luas lantai rumah = p×l = 7×6 = 42m²

Harga seluruh ubin = 105×400 = Rp.42.000
