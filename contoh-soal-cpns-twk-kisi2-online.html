13. tolong bantu kk, . latihan soal tiu cpns


Jawaban:

D. 80

Penjelasan dengan langkah-langkah:

jarak = 500 km

kecepatan = x km/jam

waktu = w + 3/4

jarak = kecepatan x waktu

500 = x (w + 3/4)

500 = xw + 3/4x ........ persamaan I

jarak = 400

kecepatan = x km/jam

w = w - 1/2

400 = x (w-1/2)

400 = xw -1/2x .............. persamaan II

persamaan I - Persamaan II (metode eliminasi xw)

100 = 3/4x - (-1/2x)

100 = 3/4x + 1/2x

100 = (3+2) /4 x

100 = 5/4 x

x = 100 x 4/5

x = 80 km/jam (D)

s1 = 500 km

w1 = t + 3/4 jam

s2 = 400 km

w2 = t - 1/2 jam

s1/w1 = s2/w2

500/(t + 3/4) = 400/(t - 1/2)

500(t - 1/2) = 400(t + 3/4)

500t - 250 = 400t + 300

500t - 400t = 300 + 250

100t = 550

t = 550/100

t = 5,5 jam

v1 = s1/w1

v1 = s1/(t + 3/4)

v1 = 500/(5,5 + 3/4)

v1 = 500/(5,5 + 0,75)

v1 = 500/6,25

v1 = 80

jadi x adalah 80 km/jam (D)

9. Sebuah pembangunan rumah akan dikerjakan oleh 10 orang selama 20 hari. Setelah bekerja 12 hari, pekerjaan dihentikan selama 3 hari.

Berapakah tambahan pekerja yang diperlukan agar pembangunan selesai tepat waktu?

P1 = 10 orang x 12 hari = 120

P2 = 0 orang x 3 hari = 0

P3 = n orang x 5 hari = 5n

Perencanaan = P1 + P2 + P3

200 = 120 + 0 + 5n

200 = 120 + 5n

pindahkan 120 ke ruas kiri sehingga menjadi -120

200 - 120 = 5n

80 = 5n

untuk mendapatkan n, bagi 80 dengan 5

n = 80 : 5

n = 16.

Pekerja tambahan = 16 orang - 10 orang

Pekerja tambahan = 6 orang
