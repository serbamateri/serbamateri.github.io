5. Tembaga mempunyai dua isotope yaitu Cu-63 Dan Cu-65 jika Ar Cu 63,5 maka kelimpahan isotope Cu-63 di Alan adalah


misal % isotop Cu-65 = x%

maka % isotop Cu-63 = 100% - x%

63,5 = 65.x% + 63 (100% - x%)
63,5 = 0,65 x + 63 - 0,63x
63,5 -63 = 0,65 x - 0,63 x 
0,5 = 0,02 x
x = 0,5/0,02 = 25

maka % isotop Cu-65 = 25 %

dan % isotop Cu-63 = 100% -25% = 75%

6. Tembaga mempunyai dua isotop yaitu cu-63 dan cu-65.jika Ar cu = 63,5; maka kelimpahan isotop cu-63 dialam adalah


misal kelimpahan Cu-63 = x%, kelimpahan Cu-65 = 100-x % 
Ar Cu = (x% * 63) + (100-x% * 65) 
63,5 = (x/100 * 63) + (100-x/100 * 65) 
63,5 = (63x + 6500 - 65x) / 100 
6350 = -2x + 6500 
2x = 6500-6350 = 150 
x = 75 

Jadi, kelimpahan Cu-63 = x% = 75%, kelimpahan Cu-65 = 100-x % = 100-75 = 25%

22. tembaga dialam mempunyai dua isotop,yaitu cu-63 sebanyak 69.09% dan cu-65 sebanyak 30.91% Hitung masa atom Cu. Masa 63 Cu dan 65 Cu adalah 62.93 dan 64.93


Jawaban:

Untuk menghitung massa atom Cu, kita dapat menggunakan rumus berikut:

Massa atom rata-rata = (Massa isotop 1 x Kelimpahan isotop 1) + (Massa isotop 2 x Kelimpahan isotop 2) + ...

Pada soal ini, kita memiliki dua isotop Cu, yaitu Cu-63 dan Cu-65, dengan kelimpahan masing-masing 69.09% dan 30.91%. Oleh karena itu, massa atom rata-rata Cu dapat dihitung sebagai berikut:

```

Massa atom rata-rata Cu = (62.93 x 0.6909) + (64.93 x 0.3091)

= 63.54

```

Jadi, massa atom Cu adalah **63.54**.

Jawaban: 63.54
