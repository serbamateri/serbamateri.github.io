22. Tes tertulis, dengan 50 butir soal, 6 soal tidak di jawab, jika seseorang memperoleh skor 171, dengan sistem ujian benar 5, salah -2, tidak menjawab 0, maka soal yang di jawab benar ada berapa butir soal?


Jawab:

Penjelasan dengan langkah-langkah:

jumlah soal = 50

soal tidak dijawab = 6 soal

misalkan :

soal dijawab dan benar = b dan skornya masing-masing +5

soal dijawab dan salah = s dan skornya masing-masing -2

soal tidak dijawab skornya masing-masing 0

b + s = 50 -6 = 44

s = 44 - b

skor :

171 = (6) (0) + (b) (5) + (s) (-2)

171 = 0 + 5b - 2s = 5b - 2 (44 - b) =  7b - 88

7b = 171 - 88 = 83

soal yang dijawab benar = b =  83 / 7 = 11, 857  (masak sih soal enggak bulat?)

52 = 58 + salah

salah = 6

= 29 + 6 + 5

= 40 soal

Uji

52 = 58 + ( -6) + 0

52 = 58 - 6

52 = 52 ( terbukti)
