24. Kalau ini jadi apa jawabannya 1. If the window is open,..... 2. You can do anything if..... 3. If you have a lot of money,.... 4. I always cry if....... 5. If l had enough money,...


Jawaban:

Contoh jawaban untuk pertanyaan tersebut adalah sebagai berikut.

If the window is open, you should close it.You can do anything if you should do it.If you have a lot of money, you should donate it. I always cry if I met my father.If I had enough money, I would build a church.

Penjelasan:

If merupakan bukti penggunaan conditional clause dalam kalimat. Kalimat seperti ini menerangkan pengandaian. Dengan kata lain terdapat hal yang diandaikan terjadi dan tindakan yang akan dilakukan jika hal tersebut sungguh terjadi.
