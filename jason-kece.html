Jawab :

KIO₃ + 5 KI + 6 HCl ⇒ 3 I₂ + 6 KCl + 3 H₂O

*mula2 :

KIO₃ = 0,5 mol

KI = 2 mol

HCl = 3 mol

I₂ = KCl = H₂O = 0 mol


*Bereaksi :

pereaksi pembatas = mol : koefisien senyawa = 2 mol KI : 5 = 0,4 mol

NB : ambil hasil bagi yang paling kecil, dalam kasus ini KI

KIO₃ = koef x pereaksi pembatas = 1 x 0,4 = 0,4 mol

KI = koef x pereaksi pembatas = 5 x 0,4 = 2 mol

HCl = koef x pereaksi pembatas = 6 x 0,4 = 2,4 mol

I₂ = H₂O = koef x pereaksi pembatas = 3 x 0,4 = 1,2 mol

KCl = koef x pereaksi pembatas = 6 x 0,4 = 2,4 mol


*Sisa :

KIO₃ = mula2 - bereaksi = 0,5 - 0,4 = 0,1 mol

KI = mula2 - bereaksi = 2 - 2 = 0 mol

HCl = mula2 - bereaksi = 3 - 2,4 = 0,6 mol

I₂ = H₂O = mula2 + bereaksi = 0 + 1,2 = 1,2 mol

KCl = mula2 + bereaksi = 0 + 2,4 = 2,4 mol


Jadi, tidak semua senyawa habis bereaksi, KIO₃ tersisa 0,1 mol dan HCl tersisa 0,6 mol

33. Jason akan berlibur keluar kota. Jarak rumah Jason ke tempat berlibur 1200 hm. Dalam perjalanan tersebut, Jason naik bus dengan kecepatan 40 km/jam. Jika Jason berangkat dari rumah pukul 10.00 WIB maka ia akan sampai di tempat berlibur adalah


Jawaban:

Jawaban

Jason akan sampai di tempat berlibur pada pukul 13.00 atau 01.00 PM

Pembahasan

S = 1.200 hm = 120 km

v = 40 km/jam

t = S/v

t = 120 km / 40 km/jam

t = 3 jam

Jason akan sampai di tempat berlibur pada pukul :

10.00 + 3 jam

= 13.00 atau 01.00 PM

Waktu Tempuh

t = s/v

t = 1.200 hm/40 km/jam

t = 120 km/40 km/jam

t = 3 jam

Waktu Tiba

= 10.00 + 3 jam

= 10.00 + 03.00

= 13.00

Jadi Jason akan tiba di tempat berlibur pada Pukul 13.00
