1. Hitung bilangan oksidasi N dalam NO3,NO2-,NH4+,NH3 dan NO2,,, !


Bilangan oksidasi N dalam NO₃⁻ ( karena tidak ada senyawa NO₃) adalah +5 . Bilangan oksidasi N dalam NO₂ ( karena tidak ada senyawa NO₂⁻ ) adalah +4 . Bilangan oksidasi N dalam NH₄⁺  adalah -3 .Bilangan oksidasi N dalam NH₃ adalah -3.Bilangan oksidasi N dalam NO₂ adalah +4.

Pembahasan

∴ Bilangan oksidasi N pada NO₃⁻ adalah +5.

Diperoleh dari atom  yang sudah diketahui bilangan oksidasinya  pada NO₃ untuk memperoleh biloks N

∵Biloks total NO₃⁻ adalah -1 ( karena bilang oksidasi suatu ion poliatomik adalah sesuai dengan jumlah muatannya ) dan bilangan oksidasi O adalah -2 ( karena bilangan oksidasi untuk Oksigen ( O) pada umumnya  adalah -2 )

NO₂ = -1

(jumlah atom N x biloks atom N) + (jumlah atom O x biloks atom O) = -1

(1 x biloks N) + (3x (-2)) = -1

Biloks N - 6 = -1

Biloks N= 6-1

Biloks N = +5

∴ Bilangan oksidasi N pada NO₂ adalah +4.

Diperoleh dari atom  yang sudah diketahui bilangan oksidasinya pada NO₂ untuk memperoleh bilangan oksidasi N

∵Bilangan oksidasi total NO₂  adalah 0 ( karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) ) dan bilangan oksidasi O adalah -2 ( karena bilangan oksidasi untuk Oksigen ( O) pada umumnya adalah -2 )

NO₂ = 0

(jumlah atom N x biloks atom N) + (jumlah atom O x biloks atom O) = 0

(1 x biloks N) + (2x (-2)) = 0

Biloks N - 4 = 0

Biloks N= 0+4

Biloks N = +4

∴ Bilangan oksidasi N pada NH₄⁺   adalah -3.

Diperoleh dari atom  yang sudah diketahui bilangan oksidasinya pada NH₄⁺ untuk memperoleh biloks N

∵Biloks total   NH₄⁺  adalah +1 (  suatu ion poliatomik adalah sesuai dengan jumlah muatannya ),  dan biloks H adalah +1 ( karena Bilangan oksidasi untuk Hidrogen ( H) adalah +1 )

Biloks  NH₄⁺   = +1

(jumlah atom N x biloks atom N) + (jumlah atom H x biloks atom H) = +1

(1 x biloks N) + ( 4 x 1) = +1

Biloks N + 4 = +1

Biloks N = +1 -4

Biloks N = -3

∴ Bilangan oksidasi N pada NH₃   adalah -3.

Diperoleh dari atom  yang sudah diketahui bilangan oksidasinya pada NH₃ untuk memperoleh biloks N

∵Biloks total   NH₃  adalah 0 (  karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) ),  dan biloks H adalah +1 ( karena Bilangan oksidasi untuk Hidrogen ( H) adalah +1 )

Biloks  NH₃   = 0

(jumlah atom N x biloks atom N) + (jumlah atom H x biloks atom H) = 0

(1 x biloks N) + ( 3 x 1) = 0

Biloks N + 3 = 0

Biloks N = -3

4. persamaan laju reaksi untuk : NH4^+ + NO2^- →N2 + 2H2O adalah r = k [NH4^+][NO^-]. Pada suhu 25°C , nilai k = 3,0 x 10^-4 mol^-1 6 detik^-1. Hitunglah laju reaksinya jika [NH4^+] = 0,25 M dan [NO2^-] = 0,02 MBantu jawab guys besok di kumpul


Laju reaksi dari persamaan v = k [NH₄⁺] [NO₂⁻] adalah 15. 10⁻⁷ M/s

Pembahasan

[tex]\boxed{\boxed{\bold{Laju~Reaksi}}}[/tex]  

Laju reaksi adalah laju berkurangnya konsentrasi reaktan (zat pereaksi) atau laju bertambahnya konsentrasi produk (zat hasil reaksi) dalam satuan waktu.

xR ----> yP

maka,

v = - [tex]\frac{\bigtriangleup R}{\bigtriangleup t}[/tex] = + [tex]\frac{\bigtriangleup P}{\bigtriangleup t}[/tex]

dengan,

R = reaktan = zat pereaksi

P = produk = zat hasil reaksi

[ΔR] = perubahan konsentrasi reaktan

[ΔP] = perubahan konsentrasi produk

- [tex]\frac{\bigtriangleup R}{\bigtriangleup t}[/tex] = laju pengurangan konsentrasi reaktan dalam satuan waktu

+ [tex]\frac{\bigtriangleup P}{\bigtriangleup t}[/tex] = laju pertambahan konsentrasi produk dalam satuan waktu

Contoh

Reaksi :

2A + B ---> 3C + 2D

Dari persamaan reaksi tersebut, 2 mol A bereaksi dengan 1 mol B yang akan membentuk 3 mol C dan 2 mol D. Hal ini berarti bahwa laju berkurangnya konsentrasi B adalah 2 kali laju berkurangnya konsentrasi A. Maka laju bertambahnya konsentrasi C adalah 3 kali laju berkurangnya konsentrasi B dan laju bertambahnya konsentrasi D adalah 2 kali laju berkurangnya konsentrasi B.

v = - ½ [tex]\frac{[\bigtriangleup A]}{\bigtriangleup t}[/tex] = - [tex]\frac{[\bigtriangleup B]}{\bigtriangleup t}[/tex] = + [tex]\frac{1}{3}[/tex] [tex]\frac{[\bigtriangleup C]}{\bigtriangleup t}[/tex] = + [tex]\frac{1}{2}[/tex] [tex]\frac{[\bigtriangleup D]}{\bigtriangleup t}[/tex]

[tex]\mathbf{Persamaan~Laju~Reaksi }[/tex]

Reaksi :

pA + qB ---> rC + sD

persamaan laju rreaksi dapat dinyatakan dengan :

[tex]v = k [A]^{x}~[B]^{y}[/tex]

v = laju reaksi (M/s)

k = tetapan laju reaksi

[A] = kosentrasi A (M)

[B] = konsentrasi B (M)

X = orde reaksi terhadap A

Y = orde reaksi terhadap B

Orde total = x + y

[NH₄⁺] = 0,25 M

[NO₂⁻] = 0,02 M

k = 3. 10⁻⁴ mol/s

14. tentukan bilangan oksidasi atom.nitrogen dalam a.Ion NO2- .B.ion NH4​


Bilangan Oksidasi Atom Nitrogen pada ion NO₂⁻ dan ion NH₄⁺ adalah .....

‖ Penyelesaian ‖

1) NO₂⁻ = bo N + 2 bo O

⇒ -1 = bo N + 2(-2)

⇒ -1 + 4 = bo N

⇒ +3 = bo N

⇒ bo N = +3 ✔

________________________

2) NH₄⁺ = bo N + 4 bo H

⇒ +1 = bo N + 4(+1)

⇒ +1 - 4 = bo N

⇒ bo N = -3 ✔

Jadi, biloks atom Nitrogen pada ion NO₂⁻ adalah +3 dan ion NH₄⁺ adalah -3.

__________________________

• Bilangan Oksidasi (biloks/bo) adalah kemampuan suatu atom memberi, melepaskan, dan menerima elektron.

• Beberapa Aturan-aturan Biloks.

1. Biloks atom dalam unsur bebas adalah 0.

2. Biloks F dalam senyawa selalu -1.

3. Biloks O adalah -2 , kecuali pada senyawa peroksida (-1) dan superoksida (-1/2).

4. Biloks H pada senyawa non logam adalah +1 dan pada senyawa logam adalah -1.

27. Bilangan oksidasi dari N dalam NO2 Cl dalam ion ClO3- N dalam ion NH4+


Bilangan oksidasi N dalam NO₂ adalah +4

Bilangan oksidasi Cl dalam ClO₃⁻ adalah +5

Bilangan oksidasi N dalam NH₄⁺ adalah -3

Pembahasan

[tex]\boxed{\boxed{\bold{Konsep~Reaksi~Oksidasi~dan~Reduksi }}}[/tex]

Terdapat tiga buah konsep reaksi oksidasi dan reduksi yaitu penggabungan dan pelepasan oksigen, pengikatan dan pelepasan elektron, dan terkakhir peningkatan dan penurunan bilangan oksidasi. Berikut ini penjelasan lengkap tentang perkembangan konsep reduksi dan oksidasi :

[tex]\mathbf{A.~Berdasarkan~Penggabungan~dan~Pelepasan~Oksigen }[/tex]

Oksidasi adalah penggabungan (pengikatan) oksigen dengan unsur/senyawa

Reduksi adalah pelepasan oksigen dari senyawanya

Contoh reaksi oksidasi :

2Mg (s) + O2 (g) --> 2MgO (s)

CH4 (g) + 2O2 (g) ---> CO2 (g) + 2H2O (g)

2CO + O2 ---> 2CO2

Zat yang memberi oksigen pada reaksi oksidasi disebut oksidator.

Contoh reaksi reduksi :

2CuO ---> 2Cu + O2

2PbO2 ---> 2PbO + O2

Zat yang menarik oksigen pada reaksi reduksi disebut reduktor.

[tex]\mathbf{B.~Berdasarkan~Pelepasan~dan~Penerimaan~Elektron }[/tex]

Oksidasi adalah pelepasan elektron (elektron terletak dikanan), sedangkan zat yang mengalami oksidasi disebut reduktor.

Reduksi adalah penerimaan elektron (elektron terletak dikiri), sedangkan zat yang mengalami reduksi disebut oksidator.

Contoh :

Reaksi antara Mg dan O2 membentuk MgO

Oksidasi : Mg ---> Mg2+ + e

Reduksi : ½ O2 + 2e ---> O2- +

Mg + ½ O2 ---> MgO

Mg mengalami oksidasi, jadi Mg adalah reduktor

O2 mengalami reduksi, jadi O2 adalah oksidator

[tex]\mathbf{C.~Berdasarkan~Perubahan~Bilangan~Oksidasi }[/tex]

Reaksi redoks adalah reaksi dimana terjadi perubahan biloks dari atom unsur sebelum dan sesudah reaksi, perubahan biloks tersebut berupa reaksi oksidasi yaitu kenaikan biloks dan reaksi reduksi yaitu penurunan biloks.

Oksidasi adalah pertambahan (kenaikan) bilangan oksidasi, sedangkan zat yang mengalami oksidasi disebut reduktor

Reduksi adalah penurunan bilangan bilangan oksidasi, sedangkan zat yang mengalami reduksi disebut oksidator

[tex]\mathbf{Aturan~Penentuan~Bilangan~Oksidasi }[/tex]

1. Bilangan okidasi atom dalam unsur bebas sama dengan 0

2. Bilangan oksidasi ion monoatom sama dengan muatan ionnya

3. Jumlah biloks dalam Senyawa sama dengan 0. Sedangkan Jumlah biloks dalam Ion Poliatom sama dengan Muatan Ionnya.

4. Biloks unsur-unsur golongan VII A (F, Cl, Br, I) dalam senyawa biner logam adalah -1

5. Biloks H jika berikatan dg non logam sama dengan +1. Sedangkan biloks H jika barikatan dg logam dan boron adalah -1.

Contoh :

Biloks H dalam HF, HCl, H2O, H2S = +1

Biloks H dalam NaH, CaH2, MgH2 = -1

6. Biloks O dalam senyawa sama dengan -2, kecuali dalam senyawa biner fluorida, peroksida, dan superoksida.

Contoh :

Biloks O dalam H2O, Na2O = -2

Biloks O dalam senyawa fluorida OF2 = +2

Biloks O dalam senyawa peroksida H2O2, Na2O2 = -1

Biloks O dalam senyawa superoksida KO2 dan CaO2 = -1/2

7. Biloks logam golongan IA (Li, Na, K, Rb, Cs, Fr) dalam senyawa sama dengan +1

8. Biloks logam golongan IIA (Be, Mg, Ca, Sr, Ba, Ra) dalam senyawa sama dengan +2

Contoh :

a. NH₄VO₂

NH₄VO₂ ---> NH₄⁺   +    VO₂⁻

VO₂⁻

1. biloks V + 2. biloks O = -1

biloks V + 2 (-2) = -1

biloks V = +4 - 1

[tex]\boxed{\boxed{\bold{biloks~V~=~+3}}}[/tex]

c. VSO₄

VSO₄  ---> Vᵃ⁺ + SO₄²⁻

biloks V + biloks SO₄²⁻ = 0

biloks V + (-2) = 0

[tex]\boxed{\boxed{\bold{biloks~V~=~+2}}}[/tex]

d. VOSO₄

VOSO₄  ---> (VO)²⁺ + SO₄²⁻

(VO)²⁺

biloks V + biloks O = +2

biloks V +  (-2) = +2

biloks V = +2 + 2

[tex]\boxed{\boxed{\bold{biloks~V~=~+4}}}[/tex]

e. VCl₃

biloks V + 3. biloks Cl = 0

biloks V + 3. (-1) = 0

[tex]\boxed{\boxed{\bold{biloks~V~=~+3}}}[/tex]

Pembahasan

Dalam reaksi redoks, terjadi oksidasi atau reduksi. Oksidasi adalah reaksi yang mengalami peningkatan bilangan oksidasi dan penurunan elektron. Sedangkan reduksi adalah reaksi yang mengalami penurunan bilangan oksidasi dan kenaikan elektron.

a. NH3 ⇒ NH4₊

biloks reaktan N = +3

biloks reaktan H = -1

biloks produk N = +5

biloks produk H = -1

Dalam reaksi ini, Nitrogen mengalami oksidasi karena biloksnya naik dari +3 menjadi +5.

b NH3 ⇒ NH2₋

biloks reaktan N = +3

biloks reaktan H = -1

biloks produk N = +1

biloks produk H = -1

Dalam reaksi ini, Nitrogen mengalami reduksi karena mengalami penurunan biloks dari +3 menjadi +1.

c. NO2 ⇒ NO3

biloks reaktan N = +4

biloks reaktan O = -2

biloks produk N = +6

biloks produk O = -2

Dalam reaksi ini, Nitrogen mengalami oksidasi karena mengalami kenaikan biloks dari +4 menjadi +6.

d. N2 ⇒ NH3

biloks reaktan N = 0

biloks produk N = +3

Setiap unsur atau molekul diatomik memiliki bilangan oksidasi 0.

Dalam reaksi ini, Nitrogen mengalami oksidasi karena mengalami kenaikan biloks dari 0 menjadi +3.

e. N2 ⇒ NO2

biloks reaktan N = 0

biloks produk N = +4

Setiap unsur atau molekul diatomik memiliki bilangan oksidasi 0.

Dalam reaksi ini, Nitrogen mengalami oksidasi karena mengalami kenaikan biloks dari 0 menjadi +4.

39. masa molekul relatif dari NH4 No2​


Penjelasan:

Untuk menghitung massa molekul relatif (MMR) dari suatu senyawa, kita perlu mengetahui massa masing-masing unsur yang membentuk senyawa tersebut dan jumlah atomnya.

1. NH4 (Ammonium):

- N (Nitrogen) memiliki massa atom sekitar 14,01 g/mol.

- H (Hidrogen) memiliki massa atom sekitar 1,01 g/mol.

Molekul NH4 terdiri dari 1 atom nitrogen dan 4 atom hidrogen.

MMR NH4 = (1 x Masa Nitrogen) + (4 x Masa Hidrogen)

MMR NH4 = (1 x 14,01 g/mol) + (4 x 1,01 g/mol)

MMR NH4 = 14,01 g/mol + 4,04 g/mol

MMR NH4 = 18,05 g/mol

2. NO2 (Nitrogen Dioksida):

- N (Nitrogen) memiliki massa atom sekitar 14,01 g/mol.

- O (Oksigen) memiliki massa atom sekitar 16,00 g/mol.

Molekul NO2 terdiri dari 1 atom nitrogen dan 2 atom oksigen.

MMR NO2 = (1 x Masa Nitrogen) + (2 x Masa Oksigen)

MMR NO2 = (1 x 14,01 g/mol) + (2 x 16,00 g/mol)

MMR NO2 = 14,01 g/mol + 32,00 g/mol

MMR NO2 = 46,01 g/mol

Jadi, massa molekul relatif (MMR) dari NH4 adalah 18,05 g/mol, dan MMR dari NO2 adalah 46,01 g/mol.
