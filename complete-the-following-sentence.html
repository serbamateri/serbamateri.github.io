Pembahasan

Present Tense untuk bentuk kalimat yang menyatakan Kegiatan berulang, rutin, kebiasaan dan hal2 bersifat umum/fakta.

Rumus Present Tense

Nominal Sentence

(+) S + Be (is/am/are) + O/C

(-) S + Be (is/am/are) + not + O/C

(?) Be (is/am/are) + S + O/C?

Verbal Sentence

(+) S + V1 (have/has) + O/C

(-) S + do/does + not + V1 (have) + O/C

(?) do/does + S + V1 (have) + O/C?

Catatan!

Jika Subjek He, She, it atau Satu nama:

kalimat positive menggunakan Verb+s/es atau HASkalimat negative dan interrogative menggunakan does

Jika Subjek I, you, we, they atau dua nama:

kalimat positive menyatakan Verb1 atau HAVE tanpa tambahan -s/eskalimat negative dan interrogative menggunakan do

26. complete the following sentence with the correct ' be ' or ' have '​


║SIMPLE PRESENT-VERBAL-NOMINAL║

Answer

1. We have a new friend. She is from Binjai.

2. My grandparents have a large Sundanese house. It has five bedrooms, two bathrooms and a store room.

3. They are Narnia and Adam. They are very good at riding horses and archery.

4. Bony has a new laptop computer. It has a great performance.

5. Do you have a lot of work to do or Are you free this afternoon, Nita?

6. Raka is my nephew. He has a twin sister.

7. Excuse me, Does your mother have my phone number? Here is my number. You can give it to her, thanks.

Discussion

Perintah pertanyaan meminta untuk mengisikan titik-titik dengan "BE" { is,am,are } and "Have."

Pembubuhan "BE" and "Have" sebagai verb { kata kerja } menyesuaikan/berdasarkan bentuk subyek kalimatnya masing-masing baik tunggal (singular noun ) maupun jamak ( plural noun ) dan konten nya apa yang di bahas. Jika, subyek kalimat nya tunggal (singular nouns) { name of person, he, she, it }, maka, gunakanlah "has." Sebaliknya, apabila subyek kalimatnya jamak ( plural nouns ) { They, we, I , you }, maka, gunakanlah "have."

1. Untuk kalimat no.1 . Kata "have" sebagai stative verb, perasaan memiliki atas teman baru, merupakan struktur kalimat verbal dan menyesuaikan subyek jamaknya "we.". Sedangkan, tobe "is" diisikan di kalimat "She is from Binjai" karena struktur kalimat nominal dan menyesuaikan subyek tunggalnya "she."

2. Untuk kalimat no.2. Kata "have" mengikuti/berdasakan subyek jamaknya "my grandparents." Sedangkan, penambahan "has" dikalimat berikutnya, menyesuaikan bentuk subyek tunggal nya "it."

3. Untuk kalimat no.3. Kalimat keduanya menggunakan tobe "are" sebagai bentuk struktur kalimat nominal dan sebagai gandengan dari subyek jamaknya "they."

4. Untuk kalimat no.4. Kata "has" mengikuti/menyesuaikan subyek tunggal nya (singular noun) "Beny." Sedangkan, untuk kalimat berikutnya menyesuaikan kata ganti subyek tunggal nya "it."

5. Untuk kalimat no.5. Ini termasuk kalimat interrogative bentuk verbal  dengan mengikuti bentuk subyek jamaknya "you". Sedangkan, untuk kalimat tanya berikutnya merupakan jenis struktur nominal sentence karena  diikuti dengan adjective "free" dan mengikuti subyek jamaknya "you".

6. Untuk kalimat no.6. Kalimat pertama merupakan kalimat nominal, tobe "is" menyesuaikan subyek tunggal nya "Raka."Sedangkan, untuk kalimat berikutnya, konten nya tentang kepemilikan, maka, menggunakan kata "has", menyesuaikan kata ganti subyek tunggalnya "he."

7. Untuk kalimat no.7. Kalimat interrogative pertama termasuk verbal. Auxiliary verb "does" merujuk pada subyek tunggal nya "your mother", maka untuk bentuk main verb-nya harus menggunakan "have". Sedangkan, kalimat berikutnya merupakan kalimat nominal. Demonstrative "here" merujuk ke bentuk subyek tunggal, maka, menggunakan tobe "is."

ExplanationSIMPLE PRESENT TENSEmerupakan tenses yang digunakan mendeskripsikan kejadian nyata di masa lampau, saat ini dan akan tetap selalu sama  di tahun - tahun mendatang.  Tenses ini juga menjabarkan fakta kebenaran umum, kebenaran yang tak terbatas dan aktivitas yang terus- menerus berulang  ( kebiasaan/habit/ daily activity )

Penggunaan keterangan frekuensi ( Adverb of frequency) sebagai identifikasi utama dalam menyatakan daily activity/habit/kebiasaan  :

Always, usually, often, everyday, seldom, rarely, daily, never, frequently, occasionally , ever, sometimes, regularly.

PATTERN  

Struktur kalimat dikategorikan ke dalam 2 jenis, Verbal dan nominal.  

1. Rumus Simple present tense ( verbal sentence )

Verbal sentence merupakan kalimat yang predikatnya berupa kata kerja aksi/tindakan. Example : eat, drink, cry, watch, wash and etc.

(+) Subject + Verb(s/es) + Object/Complement

(-)  Subject + do/does +not + Verb 1 + Object/Complement

(?) Do/does+ Subject + Verb 1 + Object/Complement?

Example :

(+ ) Mr. Jokowi has three children  ( Fakta )

( - ) Mr. Jokowi doesn't have three children

(? ) Does mr. Jokowi have three children?

2. Rumus simple present tense ( nominal sentence )

Nominal sentence merupakan kalimat yang predikatnya bukan berupa kata kerja aksi. Pada umumnya berupa kata benda ( noun), kata sifat ( adjective ), dan keterangan ( adverb )

(+ ) Subject + tobe(is/am/are) +noun/adjective/adverb

(- )  Subject + tobe(is/am/are) + not + noun/adjective/adverb

( ? ) tobe(is/am/are)+ subject +noun/adjective/adverb?

Example :

( + ) That laptop is expensive (  bagian dari deskripsi sifat )

( - )  That laptop is not expensive

( ? )  Is that laptop expensive ?
